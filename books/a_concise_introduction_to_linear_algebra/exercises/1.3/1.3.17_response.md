### Answer 1.3.17

**Given:** $P_0(5, 4, -8)$ parallel to plane $7x + y + 2z = 8$.

1.  **Find Normal:**
    $\mathbf{n} = (7, 1, 2)$.

2.  **Nonparametric Form:**
    $7(x-5) + 1(y-4) + 2(z+8) = 0$.
    $7x + y + 2z - 35 - 4 + 16 = 0$.
    $7x + y + 2z = 23$.

3.  **Parametric Form:**
    Find orthogonal vectors to $(7, 1, 2)$.
    * $\mathbf{u} = (1, -7, 0)$.
    * $\mathbf{v} = (0, -2, 1)$.
    $\mathbf{p} = (5, 4, -8) + s(1, -7, 0) + t(0, -2, 1)$.
