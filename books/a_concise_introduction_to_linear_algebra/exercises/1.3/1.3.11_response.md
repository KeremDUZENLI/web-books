### Answer 1.3.11

**Logic:**
We want to describe the triangle $ABC$.
From the previous exercise, any point on the line segment $BC$ is $(1-u)\mathbf{b} + u\mathbf{c}$.
Any point in the triangle can be found by connecting $A$ to a point on $BC$.
$\mathbf{p} = (1-v)\mathbf{a} + v [ (1-u)\mathbf{b} + u\mathbf{c} ]$.
$\mathbf{p} = (1-v)\mathbf{a} + v(1-u)\mathbf{b} + vu\mathbf{c}$.
Let $r = 1-v$, $s = v(1-u)$, $t = vu$.
Sum: $r+s+t = (1-v) + v - vu + vu = 1$.
Since $0 \le u,v \le 1$, the coefficients $r,s,t$ are non-negative.
This linear combination generates all points strictly inside or on the boundary of the triangle defined by vertices $\mathbf{a}, \mathbf{b}, \mathbf{c}$.
