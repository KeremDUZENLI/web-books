### Answer 1.3.34

**Problem:** $P_0(1, -2, 4)$ to $3x + 2y - 2z = 3$.

1.  **Normalize Equation:**
    Normal vector $\mathbf{v} = (3, 2, -2)$. Magnitude $\sqrt{17}$.
    Unit Normal $\mathbf{n} = \frac{(3, 2, -2)}{\sqrt{17}}$.
    Equation: $\frac{3x + 2y - 2z}{\sqrt{17}} = \frac{3}{\sqrt{17}}$.
    $d = 3/\sqrt{17}$.

2.  **Apply Function:**
    $f(P_0) = \mathbf{n} \cdot P_0 - d$.
    $f(P_0) = \frac{3(1) + 2(-2) - 2(4)}{\sqrt{17}} - \frac{3}{\sqrt{17}}$.
    $f(P_0) = \frac{-9}{\sqrt{17}} - \frac{3}{\sqrt{17}} = \frac{-12}{\sqrt{17}}$.

3.  **Distance:**
    Absolute value $|-12/\sqrt{17}| \approx 2.91$. (Matches 1.3.26).
