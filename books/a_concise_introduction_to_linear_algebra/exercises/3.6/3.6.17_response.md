### Answer 3.6.17

**Given:** $B = S^{-1} A S$.

**a. Transpose:**
$B^T = (S^{-1} A S)^T = S^T A^T (S^{-1})^T = S^T A^T (S^T)^{-1}$.
$B^T$ is similar to $A^T$ via matrix $S^T$.

**b. Powers:**
$B^k = (S^{-1} A S)^k = (S^{-1} A S)(S^{-1} A S)\dots(S^{-1} A S) = S^{-1} A (S S^{-1}) A \dots S = S^{-1} A^k S$.
$B^k$ is similar to $A^k$.

**c. Inverse:**
$B^{-1} = (S^{-1} A S)^{-1} = S^{-1} A^{-1} (S^{-1})^{-1} = S^{-1} A^{-1} S$.
$B^{-1}$ is similar to $A^{-1}$.
