### Answer 3.1.10

**No, this is not a vector space.**

Definition: $c(p_1, p_2) = (|c|p_1, |c|p_2)$.
Check **Distributive Law**: $(c + d)\mathbf{p} = c\mathbf{p} + d\mathbf{p}$.
Let $c = 1, d = -1$.
LHS: $(1 + (-1))\mathbf{p} = 0\mathbf{p} = (0, 0)$.
RHS: $1\mathbf{p} + (-1)\mathbf{p} = (|1|p_1, |1|p_2) + (|-1|p_1, |-1|p_2)$
$= (p_1, p_2) + (p_1, p_2) = (2p_1, 2p_2)$.
$(0,0) \neq (2p_1, 2p_2)$.
The axiom $(c+d)\mathbf{p} = c\mathbf{p} + d\mathbf{p}$ fails.
