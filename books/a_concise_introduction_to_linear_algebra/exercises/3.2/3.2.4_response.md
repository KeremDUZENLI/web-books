### Answer 3.2.4

**Set:** $U=\{x \in \mathbb{R}^4 \mid x_{1}+x_{2}+x_{3}=0\}$.

**Decision: Yes, this is a subspace.**

**Proof:**
1.  **Zero Vector:** $0+0+0=0$. In $U$.
2.  **Addition:** If $\sum u_i = 0$ and $\sum v_i = 0$, then $\sum (u_i+v_i) = \sum u_i + \sum v_i = 0 + 0 = 0$.
3.  **Scalar Mult:** If $\sum u_i = 0$, then $\sum (cu_i) = c \sum u_i = c(0) = 0$.

*Geometric Interpretation:* This is a hyperplane passing through the origin.
