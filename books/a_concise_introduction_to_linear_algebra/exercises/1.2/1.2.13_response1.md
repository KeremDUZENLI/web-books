### Answer 1.2.13

**a. Triangle Inequality:** $|\mathbf{p} + \mathbf{q}| \leq |\mathbf{p}| + |\mathbf{q}|$.
Proof:
Square the left side: $|\mathbf{p} + \mathbf{q}|^2 = |\mathbf{p}|^2 + 2\mathbf{p}\cdot\mathbf{q} + |\mathbf{q}|^2$.
Use Cauchy-Schwarz ($\mathbf{p}\cdot\mathbf{q} \leq |\mathbf{p}||\mathbf{q}|$):
$|\mathbf{p} + \mathbf{q}|^2 \leq |\mathbf{p}|^2 + 2|\mathbf{p}||\mathbf{q}| + |\mathbf{q}|^2$.
The right side is a perfect square: $(|\mathbf{p}| + |\mathbf{q}|)^2$.
So, $|\mathbf{p} + \mathbf{q}|^2 \leq (|\mathbf{p}| + |\mathbf{q}|)^2$. Take the square root.

**b. Equality Condition:**
Equality holds in step 2 only if $\mathbf{p}\cdot\mathbf{q} = |\mathbf{p}||\mathbf{q}|$.
This happens if the angle is $0^\circ$ ($\cos \theta = 1$).
Thus, vectors must be **parallel and point in the same direction**.
