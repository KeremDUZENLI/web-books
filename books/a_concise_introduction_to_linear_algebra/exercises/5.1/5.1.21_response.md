### Answer 5.1.21

**a. MATLAB:** `P = A * inv(A'*A) * A'`.
**b. Verification:**
1. `P * P - P` should be zero matrix (Idempotent).
2. `P' - P` should be zero matrix (Symmetric).
**c. Verification with Orth:**
`B = orth(A)`.
Calculate `P * eye(4)`. This gives columns of P.
Check if `rank([B P]) == rank(B)`.
Or solve `x = B \ (P(:, i))`. If residual is 0, vector is in column space.
