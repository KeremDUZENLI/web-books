### Answer 3.3.2

**Hypothesis:** Vector $\mathbf{b}$ has two decompositions: $\mathbf{b} = \sum x_i \mathbf{a}_i$ and $\mathbf{b} = \sum y_i \mathbf{a}_i$, with $\mathbf{x} \neq \mathbf{y}$.

**Goal:** Show any decomposable vector $\mathbf{c} = \sum z_i \mathbf{a}_i$ also has multiple decompositions.

**Proof:**
We can write $\mathbf{c} = \mathbf{c} + \mathbf{0} = \mathbf{c} + (\mathbf{b} - \mathbf{b})$.
$\mathbf{c} = \sum z_i \mathbf{a}_i + (\sum x_i \mathbf{a}_i - \sum y_i \mathbf{a}_i)$.
$\mathbf{c} = \sum (z_i + x_i - y_i) \mathbf{a}_i$.
Since $\mathbf{x} \neq \mathbf{y}$, at least one coefficient $x_i - y_i \neq 0$, so the new coefficients $z_i + x_i - y_i$ are different from the original $z_i$.
Thus $\mathbf{c}$ has a second, distinct decomposition.
