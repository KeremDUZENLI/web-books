### Answer 3.3.7

**Vectors:** Four vectors $\mathbf{b}, \mathbf{a}_1, \mathbf{a}_2, \mathbf{a}_3$ from Exercise 3.3.4 in $\mathbb{R}^3$.

**Statement:** **Dependent.**

**Proof:**
Any set of $m$ vectors in $\mathbb{R}^n$ where $m > n$ is linearly dependent.
Here $m=4$ vectors in $n=3$ dimensional space.
Alternatively, we showed in Ex 3.3.4 that $\mathbf{b} = 2\mathbf{a}_1 - \mathbf{a}_2 + 3\mathbf{a}_3$.
$\mathbf{b} - 2\mathbf{a}_1 + \mathbf{a}_2 - 3\mathbf{a}_3 = \mathbf{0}$ is a non-trivial relation.
