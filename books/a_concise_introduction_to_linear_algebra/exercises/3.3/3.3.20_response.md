### Answer 3.3.20

**Proof:**
Vectors $\mathbf{a}_1 \dots \mathbf{a}_m$ in $\mathbb{R}^n$. Matrix $A$ is $n \times m$.
**Span $\mathbb{R}^n \iff$ Rank $n$.**
1.  If Span $\mathbb{R}^n$: $A\mathbf{x}=\mathbf{b}$ is consistent for all $\mathbf{b}$.
    This requires a pivot in every row (to prevent $0 = \text{non-zero}$ inconsistencies).
    Number of pivots = number of rows = $n$. Rank = $n$.
    (Requires $n \le m$).
2.  If Rank $n$: There is a pivot in every row.
    The system is always consistent.
    The columns span $\mathbb{R}^n$.
