### Answer 2.3.8

**System:**
1. $2x_1 + 2x_2 - 3x_3 - 2x_4 = 4$
2. $6x_1 + 6x_2 + 3x_3 + 6x_4 = 0$

**RREF:**
1.  $R_2 \leftarrow R_2 - 3R_1$:
    LHS: $3 - 3(-3) = 12$, $6 - 3(-2) = 12$.
    RHS: $0 - 3(4) = -12$.
    Row 2: $(0, 0, 12, 12 \mid -12)$.
2.  Scale $R_2$ by $1/12$: $(0, 0, 1, 1 \mid -1)$.
3.  $R_1 \leftarrow R_1 + 3R_2$:
    $(-3 + 3, -2 + 3) \to (0, 1)$.
    RHS: $4 + 3(-1) = 1$.
    Row 1: $(2, 2, 0, 1 \mid 1)$.
4.  Scale $R_1$ by $1/2$: $(1, 1, 0, 0.5 \mid 0.5)$.

    $$ \left[ \begin{array}{cccc|c} 1 & 1 & 0 & 0.5 & 0.5 \\ 0 & 0 & 1 & 1 & -1 \end{array} \right] $$

**General Solution:**
Free vars: $x_2 = s, x_4 = t$.
$x_3 = -1 - t$.
$x_1 = 0.5 - s - 0.5t$.
$\mathbf{x} = (0.5, 0, -1, 0) + s(-1, 1, 0, 0) + t(-0.5, 0, -1, 1)$.

**Components:**
* Homogeneous $\mathbf{v} = s(-1, 1, 0, 0) + t(-0.5, 0, -1, 1)$.
* Particular $\mathbf{x}_b$ ($s=0, t=0$): $(0.5, 0, -1, 0)$.
* Particular $\mathbf{x}'_b$ ($s=1, t=0$): $(-0.5, 1, -1, 0)$.

**Equivalence:**
$\mathbf{x}'_b - \mathbf{x}_b = (-1, 1, 0, 0)$, which is part of the homogeneous solution space (specifically the $s$-vector).
Adding a homogeneous vector to a particular solution just shifts the "starting point" along the solution plane, representing the same plane.
