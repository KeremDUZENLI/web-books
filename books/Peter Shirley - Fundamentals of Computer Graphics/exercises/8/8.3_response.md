- **Line Equation:** $f(x, y) = (y_0 - y_1)x + (x_1 - x_0)y + C$.

- **Slope assumption:** $0 < m \le 1$.

- **Initialization:**

  - $x = x_0, y = y_0$.

  - Initial decision variable $d = f(x_0 + 1, y_0 + 0.5)$.

  - Let $A = y_0 - y_1$ and $B = x_1 - x_0$.

- **Incremental Update:**

  - **If $d < 0$ (Midpoint is above line $\to$ Go East):**

    - Next pixel: $(x+1, y)$

    - $d_{new} = d_{old} + A$

  - **If $d \ge 0$ (Midpoint is below line $\to$ Go North-East):**

    - Next pixel: $(x+1, y+1)$

    - $d_{new} = d_{old} + A + B$

- **Note:** $A$ and $B$ are constants computed once at the start.
