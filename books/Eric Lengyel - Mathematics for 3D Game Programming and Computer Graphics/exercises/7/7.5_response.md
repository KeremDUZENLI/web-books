### Answer 7.5

To implement this program, follow these logical steps:

1.  **Initialize Arrays**: Create arrays to store accumulated normals and tangents for each vertex, initialized to zero vector $\langle 0,0,0 \rangle$.

2.  **Iterate Triangles**: For each triangle with indices $(i_0, i_1, i_2)$:
    * Get positions $P_0, P_1, P_2$ and texture coords $(u_0, v_0), (u_1, v_1), (u_2, v_2)$.
    * **Calculate Face Normal**: $N_{face} = (P_1 - P_0) \times (P_2 - P_0)$.
    * **Calculate Face Tangent**:
        * Compute edge vectors $Q_1 = P_1 - P_0$, $Q_2 = P_2 - P_0$.
        * Compute delta UVs $(s_1, t_1) = (u_1-u_0, v_1-v_0)$, $(s_2, t_2) = (u_2-u_0, v_2-v_0)$.
        * Solve linear system for Tangent $T$:
            $$T = \frac{1}{s_1 t_2 - s_2 t_1} (t_2 Q_1 - t_1 Q_2)$$
    * **Accumulate**: Add $N_{face}$ to the normal accumulator for vertices $i_0, i_1, i_2$. Add $T$ to the tangent accumulator for vertices $i_0, i_1, i_2$.

3.  **Normalize**: Iterate through all vertices.
    * Normalize the accumulated normal vector $N_{v}$.
    * Gram-Schmidt Orthogonalize the tangent $T_{acc}$ against $N_{v}$: $T_{v} = (T_{acc} - (N_{v} \cdot T_{acc})N_{v})$.
    * Normalize $T_{v}$.

4.  **Output**: The resulting arrays contain the vertex normals and tangents.
