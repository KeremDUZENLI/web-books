### Answer 7.1

The attenuation formula for a point light is:
$$I(d) = \frac{I_0}{k_c + k_l d + k_q d^2}$$

Given constants $k_c=1, k_l=0, k_q=0.5$, the intensity at distance $d$ is:
$$I(d) = \frac{I_0}{1 + 0.5d^2}$$

**1. Calculate intensity at $d=1$:**
$$I(1) = \frac{I_0}{1 + 0.5(1)^2} = \frac{I_0}{1.5}$$

**2. Set up equation for target intensity:**
We require the intensity at distance $d$ to be one-fourth of $I(1)$:
$$I(d) = \frac{1}{4} I(1) = \frac{1}{4} \left( \frac{I_0}{1.5} \right) = \frac{I_0}{6}$$

**3. Solve for $d$:**
$$\frac{I_0}{1 + 0.5d^2} = \frac{I_0}{6}$$
$$1 + 0.5d^2 = 6$$
$$0.5d^2 = 5$$
$$d^2 = 10$$
$$d = \sqrt{10} \approx 3.16 \text{ meters}$$
