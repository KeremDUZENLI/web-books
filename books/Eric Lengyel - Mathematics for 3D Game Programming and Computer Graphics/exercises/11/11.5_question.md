### Answer 11.5

Curve $Q(t) = \langle t, t^2, 0 \rangle$.
This is a parabola in the xy-plane.

**Formula for Curvature $\kappa(t)$:**
$$\kappa(t) = \frac{||Q'(t) \times Q''(t)||}{||Q'(t)||^3}$$

**Derivatives:**
Velocity $Q'(t) = \langle 1, 2t, 0 \rangle$.
Acceleration $Q''(t) = \langle 0, 2, 0 \rangle$.

**Cross Product:**
$Q' \times Q'' = \begin{vmatrix} i & j & k \\ 1 & 2t & 0 \\ 0 & 2 & 0 \end{vmatrix} = i(0) - j(0) + k(2 - 0) = \langle 0, 0, 2 \rangle$.
Magnitude $||Q' \times Q''|| = 2$.

**Speed:**
$||Q'(t)|| = \sqrt{1^2 + (2t)^2 + 0} = \sqrt{1 + 4t^2}$.

**Curvature Function:**
$$\kappa(t) = \frac{2}{(\sqrt{1 + 4t^2})^3} = \frac{2}{(1 + 4t^2)^{3/2}}$$

**Evaluation:**
**At $t=0$:**
$\kappa(0) = \frac{2}{(1 + 0)^{3/2}} = \frac{2}{1} = 2$.

**At $t=1$:**
$\kappa(1) = \frac{2}{(1 + 4)^{3/2}} = \frac{2}{5^{3/2}} = \frac{2}{5\sqrt{5}} = \frac{2\sqrt{5}}{25} \approx 0.1789$.
