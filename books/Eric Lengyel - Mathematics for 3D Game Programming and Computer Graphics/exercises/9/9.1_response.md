### Answer 9.1

We need to modify the projection matrix entry $M_{33}$ such that the depth value at $z = -20$ equals the depth value of a point at $z_{target} = -20 + 0.2 = -19.8$ using the unmodified matrix.

**1. Standard Depth Formula:**
The depth $d$ (NDC z-coordinate) is given by:
$$d(z) = M_{33} + \frac{M_{34}}{z}$$
For standard OpenGL projection:
$$M_{33} = -\frac{f+n}{f-n} = -\frac{100+1}{100-1} = -\frac{101}{99} \approx -1.020202$$
$$M_{34} = -\frac{2fn}{f-n} = -\frac{2(100)(1)}{100-1} = -\frac{200}{99} \approx -2.020202$$

**2. Calculate Target Depth:**
We want the depth at $z=-20$ to look like the depth at $z=-19.8$ with the original matrix.
$$d_{target} = d(-19.8) = -\frac{101}{99} + \frac{-200/99}{-19.8}$$
$$d_{target} = -\frac{101}{99} + \frac{200}{99 \times 19.8} = \frac{-101(19.8) + 200}{99(19.8)} = \frac{-1999.8 + 200}{1960.2} = \frac{-1799.8}{1960.2} \approx -0.9181716$$

**3. Set up Modified Equation:**
Let the modified entry be $M'_{33} = k \cdot M_{33}$.
We want the depth at the original $z = -20$ using $M'_{33}$ to equal $d_{target}$.
$$d_{mod}(-20) = M'_{33} + \frac{M_{34}}{-20} = d_{target}$$
$$k \left(-\frac{101}{99}\right) + \frac{-200/99}{-20} = d_{target}$$
$$k \left(-\frac{101}{99}\right) + \frac{10}{99} = d_{target}$$

**4. Solve for k:**
$$k \left(-\frac{101}{99}\right) = d_{target} - \frac{10}{99}$$
$$d_{target} \approx -0.9181716$$
$$\frac{10}{99} \approx 0.1010101$$
$$RHS \approx -0.9181716 - 0.1010101 = -1.0191817$$

$$k = \frac{-1.0191817}{-101/99} = \frac{-1.0191817}{-1.020202}$$
$$k \approx 0.99900$$

**Exact Calculation:**
$$k M_{33} = M_{33} + M_{34} \left( \frac{1}{z_{target}} - \frac{1}{z_{original}} \right)$$
$$k = 1 + \frac{M_{34}}{M_{33}} \left( \frac{1}{-19.8} - \frac{1}{-20} \right)$$
Ratio $\frac{M_{34}}{M_{33}} = \frac{-2fn/(f-n)}{-(f+n)/(f-n)} = \frac{2fn}{f+n} = \frac{200}{101}$.
Difference term: $\frac{1}{-19.8} - \frac{1}{-20} = -0.050505 + 0.050000 = -0.000505$.
$$k = 1 + \frac{200}{101} (-0.000505) \approx 1 - 0.001 = 0.999$$

The multiplier is approximately **0.999**.
