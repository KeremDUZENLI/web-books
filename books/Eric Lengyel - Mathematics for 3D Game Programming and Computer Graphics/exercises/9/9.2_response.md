### Answer 9.2

We need to find the offset distance $\Delta z$ that corresponds to the minimum resolvable depth difference in a 16-bit depth buffer at $z = -20$.

**1. Depth Buffer Precision:**
A 16-bit depth buffer has $2^{16} = 65536$ discrete values.
Assuming the depth range is mapped to $[0, 1]$ (or $[-1, 1]$), the smallest change in the normalized depth $d$ is:
$$\Delta d = \frac{1}{65536} \times (\text{Range})$$
For OpenGL standard $[-1, 1]$, Range = 2.
$$\Delta d = \frac{2}{65536} \approx 3.05 \times 10^{-5}$$

**2. Derivative of Depth Function:**
The depth function is $d(z) = A + \frac{B}{z}$, where $B = -\frac{2fn}{f-n}$.
$$d'(z) = -\frac{B}{z^2}$$
We can approximate $\Delta d \approx |d'(z)| \Delta z$.
$$\Delta z \approx \frac{\Delta d}{|d'(z)|} = \frac{\Delta d}{|-B/z^2|} = \Delta d \frac{z^2}{|B|}$$

**3. Calculate Values:**
$n=1, f=100$.
$B = -\frac{2(100)(1)}{99} = -\frac{200}{99} \approx -2.02$.
$z = -20, z^2 = 400$.

**4. Solve for $\Delta z$:**
$$\Delta z = (3.05 \times 10^{-5}) \frac{400}{2.02}$$
$$\Delta z = \frac{0.0122}{2.02}$$
$$\Delta z \approx 0.006$$

The least distance is approximately **0.006** units.
