**RK4 Implementation for $y' = f(x, y)$:**

```cpp
float RK4(float x, float y, float h, float (*f)(float, float))
{
    float k1 = f(x, y);
    float k2 = f(x + 0.5f * h, y + 0.5f * h * k1);
    float k3 = f(x + 0.5f * h, y + 0.5f * h * k2);
    float k4 = f(x + h, y + h * k3);

    return y + (h / 6.0f) * (k1 + 2.0f * k2 + 2.0f * k3 + k4);
}
```
