### Answer 4.2

Rotation angle $\theta = 60^\circ$. Axis vector $V = (0, 3, 4)$.

**1. Normalize the axis**
Length $||V|| = \sqrt{0^2 + 3^2 + 4^2} = \sqrt{0 + 9 + 16} = \sqrt{25} = 5$.
Normalized axis $A = \frac{V}{||V||} = \langle 0, \frac{3}{5}, \frac{4}{5} \rangle$.

**2. Calculate Quaternion Components**
Formula: $q = \langle \cos\frac{\theta}{2}, A \sin\frac{\theta}{2} \rangle$
Half angle $\frac{\theta}{2} = 30^\circ$.
$s = \cos(30^\circ) = \frac{\sqrt{3}}{2}$.
Factor for vector part: $\sin(30^\circ) = \frac{1}{2}$.

Vector part $v = A \sin(30^\circ) = \langle 0, \frac{3}{5}, \frac{4}{5} \rangle \cdot \frac{1}{2} = \langle 0, \frac{3}{10}, \frac{2}{5} \rangle$.

**3. Result**
$$q = \langle \frac{\sqrt{3}}{2}, 0, \frac{3}{10}, \frac{2}{5} \rangle$$
(Written as $\langle w, x, y, z \rangle$ or $s + v$).
