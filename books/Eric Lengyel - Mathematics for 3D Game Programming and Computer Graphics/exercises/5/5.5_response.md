### Answer 5.5

Given:
Horizontal FOV $\alpha = 90^\circ$.
Aspect ratio $a = 0.75$.
(Assume standard camera looking down $-z$ axis).

**1. Calculate Frustum Dimensions at $z = -1$:**
$$x_{right} = \tan\left(\frac{90^\circ}{2}\right) = \tan(45^\circ) = 1$$
$$x_{left} = -1$$
$$y_{top} = \frac{x_{right}}{a} = \frac{1}{0.75} = \frac{4}{3} \approx 1.333$$
$$y_{bottom} = -\frac{4}{3} \approx -1.333$$

**2. Determine Plane Equations (Normal pointing inward):**
General plane through origin containing line $x = c z$: Normal is orthogonal to direction.

* **Right Plane:** Passes through $(0,0,0)$ and $(1, 0, -1)$.
    Normal $N_R$ must be perpendicular to edge vector $E = (1, 0, -1)$.
    $N_R = \langle -1, 0, -1 \rangle$ (normalized: $\langle -\frac{1}{\sqrt{2}}, 0, -\frac{1}{\sqrt{2}} \rangle$).
    Equation: $-x - z = 0$ (or $x + z = 0$).

* **Left Plane:** Passes through $(0,0,0)$ and $(-1, 0, -1)$.
    Normal $N_L$ perpendicular to $(-1, 0, -1)$.
    $N_L = \langle 1, 0, -1 \rangle$ (normalized: $\langle \frac{1}{\sqrt{2}}, 0, -\frac{1}{\sqrt{2}} \rangle$).
    Equation: $x - z = 0$.

* **Top Plane:** Passes through $(0,0,0)$ and $(0, 1.333, -1)$.
    Vector $V = (0, 4/3, -1)$.
    Normal $N_T$ is perpendicular to $V$, pointing down.
    $N_T = \langle 0, -1, -\frac{4}{3} \rangle$.
    Equation: $-y - 1.333z = 0$.

* **Bottom Plane:** Passes through $(0,0,0)$ and $(0, -1.333, -1)$.
    Normal $N_B$ points up.
    $N_B = \langle 0, 1, -\frac{4}{3} \rangle$.
    Equation: $y - 1.333z = 0$.
