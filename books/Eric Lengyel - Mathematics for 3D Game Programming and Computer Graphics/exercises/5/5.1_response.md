### Answer 5.1

Let the points be $P_1 = (1, 2, 0)$, $P_2 = (2, 0, -1)$, and $P_3 = (3, -2, -1)$.

**1. Calculate vectors lying on the plane:**
$$V_1 = P_2 - P_1 = \langle 2-1, 0-2, -1-0 \rangle = \langle 1, -2, -1 \rangle$$
$$V_2 = P_3 - P_1 = \langle 3-1, -2-2, -1-0 \rangle = \langle 2, -4, -1 \rangle$$

**2. Calculate normal vector $N$ using cross product:**
$$N = V_1 \times V_2 = \begin{vmatrix} i & j & k \\ 1 & -2 & -1 \\ 2 & -4 & -1 \end{vmatrix}$$
$$x = (-2)(-1) - (-1)(-4) = 2 - 4 = -2$$
$$y = (-1)(2) - (1)(-1) = -2 + 1 = -1$$
$$z = (1)(-4) - (-2)(2) = -4 + 4 = 0$$
$$N = \langle -2, -1, 0 \rangle$$

**3. Calculate plane constant D:**
Using the plane equation $N \cdot P + D = 0$:
$$\langle -2, -1, 0 \rangle \cdot \langle 1, 2, 0 \rangle + D = 0$$
$$(-2)(1) + (-1)(2) + (0)(0) + D = 0$$
$$-2 - 2 + D = 0 \implies D = 4$$

**4. Result:**
The 4D vector $(N, D)$ is $\langle -2, -1, 0, 4 \rangle$.
(Note: Often normalized, but not required by the question phrasing).
