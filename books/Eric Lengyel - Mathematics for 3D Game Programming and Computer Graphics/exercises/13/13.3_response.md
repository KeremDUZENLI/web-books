### Answer 13.3

**1. Initial Vertical Velocity**
$v_z = v_{initial} \sin(45^\circ) = 20 (\frac{\sqrt{2}}{2}) = 10\sqrt{2} \approx 14.14$ m/s.
Initial height $z_0 = 20$ m.

**2. Maximum Height Formula**
$h_{max} = z_0 + \frac{v_z^2}{2g}$
$v_z^2 = (10\sqrt{2})^2 = 200$.
$$h_{max} = 20 + \frac{200}{2g} = 20 + \frac{100}{g}$$

Using $g \approx 9.8$ m/sÂ²:
$$h_{max} = 20 + \frac{100}{9.8} \approx 20 + 10.2 = 30.2 \text{ meters}$$
