### Answer 13.2

Equation: $x''(t) + 16x(t) = 0$.
Initial conditions: $x(0)=3, x'(0)=1$.

**1. General Solution**
Auxiliary equation: $r^2 + 16 = 0 \implies r = \pm 4i$.
$\alpha = 0, \beta = 4$.
$$x(t) = A \cos(4t) + B \sin(4t)$$

**2. Derivative**
$$x'(t) = -4A \sin(4t) + 4B \cos(4t)$$

**3. Apply Initial Conditions**
$x(0) = A(1) + B(0) = 3 \implies A = 3$.
$x'(0) = -4(3)(0) + 4B(1) = 1 \implies 4B = 1 \implies B = 1/4$.

**4. Final Solution**
$$x(t) = 3 \cos(4t) + \frac{1}{4} \sin(4t)$$
