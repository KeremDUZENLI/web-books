### Answer 8.3

Formula for effective radius:
$$r_{eff} = \frac{1}{2} (|R \cdot N| + |S \cdot N| + |T \cdot N|)$$

Given:
$R = \langle 2, 0, 1 \rangle$
$S = \langle 1, 0, -2 \rangle$
$T = \langle 0, 1, 0 \rangle$
$N = \langle \frac{1}{\sqrt{3}}, -\frac{1}{\sqrt{3}}, \frac{1}{\sqrt{3}} \rangle$

**1. Calculate Dot Products:**
$$R \cdot N = 2(\frac{1}{\sqrt{3}}) + 0 + 1(\frac{1}{\sqrt{3}}) = \frac{3}{\sqrt{3}} = \sqrt{3}$$
$$S \cdot N = 1(\frac{1}{\sqrt{3}}) + 0 - 2(\frac{1}{\sqrt{3}}) = -\frac{1}{\sqrt{3}}$$
$$T \cdot N = 0 + 1(-\frac{1}{\sqrt{3}}) + 0 = -\frac{1}{\sqrt{3}}$$

**2. Calculate Magnitude Sum:**
$$|R \cdot N| = \sqrt{3}$$
$$|S \cdot N| = \frac{1}{\sqrt{3}}$$
$$|T \cdot N| = \frac{1}{\sqrt{3}}$$

Sum $= \sqrt{3} + \frac{2}{\sqrt{3}} = \frac{3}{\sqrt{3}} + \frac{2}{\sqrt{3}} = \frac{5}{\sqrt{3}}$

**3. Calculate $r_{eff}$:**
$$r_{eff} = \frac{1}{2} \left( \frac{5}{\sqrt{3}} \right) = \frac{5}{2\sqrt{3}} = \frac{5\sqrt{3}}{6}$$
