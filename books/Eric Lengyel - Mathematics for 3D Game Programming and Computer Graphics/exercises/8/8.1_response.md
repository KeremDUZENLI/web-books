### Answer 8.1

Let $d = ||Q_2 - Q_1||$ be the distance between the centers.

**Case 1: One sphere encloses the other**
If $d + r_2 \le r_1$, then $S_1$ encloses $S_2$.
$$Q = Q_1$$
$$r = r_1$$
If $d + r_1 \le r_2$, then $S_2$ encloses $S_1$.
$$Q = Q_2$$
$$r = r_2$$

**Case 2: Disjoint or Intersecting (neither encloses the other)**
The bounding sphere must span the extremum points along the line connecting the centers.
Unit vector $V = \frac{Q_2 - Q_1}{d}$.
Extremum of $S_1$: $P_1 = Q_1 - r_1 V$.
Extremum of $S_2$: $P_2 = Q_2 + r_2 V$.

Diameter $D = ||P_2 - P_1|| = ||(Q_2 + r_2 V) - (Q_1 - r_1 V)|| = ||(Q_2 - Q_1) + (r_1 + r_2)V|| = ||dV + (r_1+r_2)V|| = d + r_1 + r_2$.

Radius $r$:
$$r = \frac{D}{2} = \frac{d + r_1 + r_2}{2}$$

Center $Q$:
$$Q = \frac{P_1 + P_2}{2} = \frac{Q_1 - r_1 V + Q_2 + r_2 V}{2}$$
Substitute $V = \frac{Q_2 - Q_1}{d}$:
$$Q = \frac{Q_1 + Q_2}{2} + \frac{r_2 - r_1}{2} \left( \frac{Q_2 - Q_1}{d} \right)$$
