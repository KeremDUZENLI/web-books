### Answer 8.2

The cone has apex at $A = (0, 0, h)$ and base rim points $B$ satisfying $x^2 + y^2 = s^2, z=0$.

**Case 1: $s < h$ (Tall Cone)**
The optimal sphere passes through the apex and the base rim.
Let center be $Q = (0, 0, z_c)$ and radius be $r$.
Distance to apex: $h - z_c = r \implies z_c = h - r$.
Distance to rim point $(s, 0, 0)$: $\sqrt{s^2 + z_c^2} = r$.

Substitute $z_c$:
$$s^2 + (h - r)^2 = r^2$$
$$s^2 + h^2 - 2hr + r^2 = r^2$$
$$s^2 + h^2 = 2hr$$
$$r = \frac{s^2 + h^2}{2h}$$

Center $Q$:
$$z_c = h - \frac{s^2 + h^2}{2h} = \frac{2h^2 - s^2 - h^2}{2h} = \frac{h^2 - s^2}{2h}$$
$$Q = \langle 0, 0, \frac{h^2 - s^2}{2h} \rangle$$

**Case 2: $s \ge h$ (Flat Cone)**
The optimal sphere is the bounding sphere of the base circle alone, which also encloses the apex (since height $h$ is less than radius $s$).
$$r = s$$
$$Q = \langle 0, 0, 0 \rangle$$
