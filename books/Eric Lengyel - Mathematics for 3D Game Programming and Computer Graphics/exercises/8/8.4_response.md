### Answer 8.4

**Bounding Box Construction:**
1.  Compute Mean $m = \frac{1}{N}\sum P_i$.
2.  Compute Covariance Matrix $C = \frac{1}{N}\sum (P_i - m)(P_i - m)^T$.
3.  Compute eigenvectors of $C$ to find principal axes $R, S, T$.
4.  Project all $P_i$ onto $R, S, T$ to find min/max extents.
5.  Construct box planes from extents.

**Bounding Sphere Construction:**
1.  Identify principal axis $R$.
2.  Find points $P_k, P_l$ with min/max dot products along $R$.
3.  Initial Center $Q = (P_k + P_l) / 2$; Radius $r = ||P_k - Q||$.
4.  Iterate all $P_i$:
    If $||P_i - Q|| > r$:
    $G = Q - r \frac{P_i - Q}{||P_i - Q||}$
    $Q_{new} = (G + P_i) / 2$
    $r_{new} = ||P_i - Q_{new}||$

**Bounding Ellipsoid Construction:**
1.  Calculate bounding box extents $a, b, c$ along natural axes $R, S, T$.
2.  Scale vertices: $P'_i = M P_i$ where $M$ scales by $(1/a, 1/b, 1/c)$ along axes.
3.  Compute bounding sphere for $P'_i$ (center $Q'$, radius $r'$).
4.  Ellipsoid center $Q = M^{-1} Q'$.
5.  Semiaxes are $r'a, r'b, r'c$.

**Bounding Cylinder Construction:**
1.  Project points onto plane perpendicular to principal axis $R$: $H_i = P_i - (P_i \cdot R)R$.
2.  Compute bounding circle for $H_i$ (Center $Q_{circle}$, radius $r$).
3.  Find min/max projections along $R$: $min_R, max_R$.
4.  Cylinder endpoints: $Q_1 = Q_{circle} + min_R R$, $Q_2 = Q_{circle} + max_R R$.
