### Answer 6.3

We want to find the p-th root of $r$. Let $x = r^{1/p}$.
This is equivalent to finding the root of the function:
$$f(x) = x^p - r = 0$$

**1. Compute Derivative:**
$$f'(x) = p x^{p-1}$$

**2. Apply Newton's Method Formula:**
$$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$$
$$x_{n+1} = x_n - \frac{x_n^p - r}{p x_n^{p-1}}$$

**3. Simplify:**
$$x_{n+1} = x_n - \frac{x_n^p}{p x_n^{p-1}} + \frac{r}{p x_n^{p-1}}$$
$$x_{n+1} = x_n - \frac{1}{p} x_n + \frac{r}{p} x_n^{1-p}$$
$$x_{n+1} = \frac{p-1}{p} x_n + \frac{r}{p} x_n^{1-p}$$
