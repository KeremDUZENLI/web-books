### Exercise 2.1

Performing the scale before the translate is one way to accomplish the display transformation; however, the reverse order will work as well, with different values for the numeric properties.

Using the laboratory, visit V.03 and edit the XAML code to reverse the order of the two transforms. First, change the order without adjusting the numeric properties, notice how the rendered scene changes, and then change the properties as needed to restore the desired target rendering.
