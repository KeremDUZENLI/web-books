### Answer 2.6

**1. Define Template:**

Create a thinner polygon (e.g., width 0.1) in `Canvas.Resources`.

```xml
<ControlTemplate x:Key="SecondHandTemplate">
    <Polygon Points="-0.1,2 -0.1,9 0,10 0.1,9 0.1,2" Fill="Red"/>
</ControlTemplate>

```

**2. Instantiate:**

Add a control referencing the template with a RotateTransform (e.g., 15 seconds = 90 degrees).

```xml
<Control Template="{StaticResource SecondHandTemplate}">
    <Control.RenderTransform>
        <RotateTransform Angle="90"/>
    </Control.RenderTransform>
</Control>
```
