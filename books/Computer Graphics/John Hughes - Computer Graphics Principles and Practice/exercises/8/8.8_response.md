### Answer 8.8

**Implementation Logic:**
1.  **State:** `List<Vertex> vertices`, `List<Edge> edges`, `List<Triangle> triangles`. `Vertex selectedVertex = null`. `List<Vertex> controlSelection`.
2.  **Input Handling:**
    * **Shift-Click (Edge Creation):**
        * Hit test to find clicked vertex $v_{target}$.
        * If $v_{target}$ exists:
            * If `selectedVertex` is not null:
                * Check if edge $(selectedVertex, v_{target})$ exists.
                * If yes, remove it.
                * If no, add new Edge.
                * `selectedVertex` = $v_{target}$ (chaining).
            * Else: `selectedVertex` = $v_{target}$.
        * Else (Click on empty space):
            * Create new vertex $v_{new}$.
            * If `selectedVertex` is not null, add edge $(selectedVertex, v_{new})$.
            * `selectedVertex` = $v_{new}$.
    * **Control-Click (Triangle Creation):**
        * Hit test for vertex $v$.
        * If found:
            * Add $v$ to `controlSelection`.
            * If `controlSelection.Count == 3`:
                * Check if Triangle(v1, v2, v3) exists.
                * If yes, delete it.
                * If no, create new Triangle.
                * Clear `controlSelection`.
    * **Redraw:**
        * Draw all edges.
        * Draw triangles (filled semi-transparent).
        * Draw vertices (highlight `selectedVertex` and `controlSelection` members).
