### Answer 8.5

**Data Structure Assumption (Winged-Edge):**
* Face `F` stores a reference to one edge `e` that belongs to it.
* Edge `e` stores `e.left_face`, `e.right_face`, `e.next_left` (next edge around left face), `e.next_right` (next edge around right face).

**Pseudocode:**

```

function GetFaceEdges(Face f):
List edges = []
Edge start_edge = f.edge

```
if start_edge is null:
    return edges
    
Edge current = start_edge

do:
    edges.add(current)
    
    // Determine which side of the edge the face f is on
    if current.left_face == f:
        current = current.next_left
    else if current.right_face == f:
        current = current.next_right
    else:
        Error("Face pointer mismatch")
        
while (current != start_edge)

return edges

