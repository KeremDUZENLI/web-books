### Answer 8.6

**(a) Algorithm:**
1.  Initialize a dictionary `orientation` mapping each triangle to `+1` (default) or `-1` (flipped). Set `visited` set to empty.
2.  Pick an arbitrary starting triangle $T_0$. Mark as visited.
3.  Push $T_0$ onto a stack.
4.  While stack is not empty:
    * Pop $T_{curr}$.
    * For each neighbor $T_{next}$ sharing edge $(u, v)$:
        * Check the edge order in $T_{next}$.
        * If $T_{curr}$ has edge $(u, v)$ and $T_{next}$ has $(v, u)$, they are consistent.
        * If $T_{next}$ has $(u, v)$, they are inconsistent.
        * If $T_{next}$ is not visited:
            * Set $T_{next}$ orientation to match $T_{curr}$.
            * Mark visited and push to stack.
        * If $T_{next}$ IS visited:
            * Check if its existing orientation is consistent with $T_{curr}$.
            * If inconsistent, **return False** (M is non-orientable, e.g., Möbius strip).
5.  **Return True**.

**(b) At most two orientations:**
Once the orientation of the first triangle is fixed (e.g., normal points "out"), the orientation of every adjacent triangle is strictly constrained by the consistency requirement (shared edges must go in opposite directions). This constraint propagates deterministically through the connected mesh. Therefore, there are no degrees of freedom left after the first choice.

**(c) Exactly two orientations:**
If the mesh is orientable, the initial choice (Step 2) could be "Clockwise" or "Counter-clockwise". Since the consistency propagation is valid (by definition of orientable), choosing the opposite for $T_0$ effectively flips the orientation of every triangle in the mesh. Thus, exactly two global solutions exist: "All normals outward" and "All normals inward".

**(d) Disconnected Mesh:**
For a mesh with $k$ connected components, each component can be oriented independently in 2 ways. The total combinations are $2 \times 2 \times \dots \times 2 = 2^k$ orientations.
