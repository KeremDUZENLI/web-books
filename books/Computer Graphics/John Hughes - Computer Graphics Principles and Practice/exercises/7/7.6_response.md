### Answer 7.6

(a) If we multiply the coefficients $(A, B, C)$ by $\lambda \ne 0$, the equation becomes $\lambda A x + \lambda B y + \lambda C = 0$. Dividing by $\lambda$ returns the original equation, so the set of solutions (the line) is unchanged.
(b) The "normal form" divides $(A, B, C)$ by $\sqrt{A^2+B^2+C^2}$. If we start with $(\lambda A, \lambda B, \lambda C)$ where $\lambda > 0$, the divisor is $\lambda\sqrt{A^2+B^2+C^2}$. The ratio is $\frac{\lambda A}{\lambda\sqrt{\dots}} = \frac{A}{\sqrt{\dots}}$, which is the same normal form.
(c) The vector $n=[A, B]^T$ is the normal to the line. If $A^2+B^2=0$, then $A=0$ and $B=0$, so the equation is $C=0$, which is either always true or always false (not a line). Thus, for a valid line, the magnitude is never zero.
