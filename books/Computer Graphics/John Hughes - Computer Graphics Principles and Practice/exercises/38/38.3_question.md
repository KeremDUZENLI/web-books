### Exercise 38.3
    
Look for performance dependencies on data coherence, using the framework developed in the preceding exercise. Using linear image interpolation, map a 1,024 x 1,024-texture image to a $256 \times 256$ triangle. Compare the performance of 1-to-1, 2-to-1, and 4-to-1 texel-to-pixel mappings. If there is no performance difference, increase the demand for memory bandwidth by utilizing multiple textures until a difference is found.
