### Exercise 4.5

The scene currently has one light. Add a second light source to the scene.

1.  Create a `PointLight` with a distinct color (e.g., Red or Blue).
2.  Create variables `_lightX`, `_lightY`, `_lightZ` to hold its position.
3.  In `OnUpdate`, update the light's position using these variables.
4.  Add sliders for these variables to the Tweak Bar and observe the lighting interactions on the robot.
