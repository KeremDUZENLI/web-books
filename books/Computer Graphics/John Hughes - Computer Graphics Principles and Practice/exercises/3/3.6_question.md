### Exercise 3.6

Using the `Sphere` primitive, build a snowman.

You will need three spheres of decreasing size.

1.  Create a "base" sphere at the origin with a scale of 1.0 (or radius 3).
2.  Create a "torso" sphere, scaled smaller, and translate it up the Y-axis so it sits on the base.
3.  Create a "head" sphere, scaled even smaller, and translate it further up.
4.  (Optional) Add small black spheres for eyes.
