**Robot Geometry:**

1.  **Torso:** Cube at origin. Scale $(2, 3, 1)$.
2.  **Head:** Cube. Scale $(1, 1, 1)$. Translate $(0, 2, 0)$ (Relative to torso top).
3.  **Arms (x2):**
    * Scale $(0.5, 2.5, 0.5)$.
    * Translate Left: $(-1.5, 0, 0)$.
    * Translate Right: $(1.5, 0, 0)$.
4.  **Legs (x2):**
    * Scale $(0.6, 3.0, 0.6)$.
    * Translate Left: $(-0.6, -3, 0)$.
    * Translate Right: $(0.6, -3, 0)$.

**Grouping:**
Add all `GeometryModel3D` objects to a `Model3DGroup`. This allows applying a single transform to the group to move the whole robot.
