### Answer 13.2
We analyze the function $f(z) = \frac{n+f}{n-f} + \frac{2nf}{n-f} \frac{1}{z}$. (Note: The prompt text uses the formulation mapping $[n, f]$ to $[-1, 1]$ or similar).
Let's check the values given in the standard projection matrix formula:
$z' = -\frac{f+n}{f-n} z - \frac{2fn}{f-n}$ (numerator after persp division by $-z$).
After dividing by $w = -z$, the depth value is $d(z) = \frac{f+n}{f-n} + \frac{2fn}{f-n}\frac{1}{z}$.

Check $z = -n$:
$d(-n) = \frac{f+n}{f-n} + \frac{2fn}{f-n}\frac{1}{-n} = \frac{f+n - 2f}{f-n} = \frac{n-f}{f-n} = -1$.

Check $z = -f$:
$d(-f) = \frac{f+n}{f-n} + \frac{2fn}{f-n}\frac{1}{-f} = \frac{f+n - 2n}{f-n} = \frac{f-n}{f-n} = 1$.

The values are verified.
