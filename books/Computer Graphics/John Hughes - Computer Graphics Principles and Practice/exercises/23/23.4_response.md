### Answer 23.4
    
**Partial Derivatives:**
Given $S(u,v) = (1-u)(1-v)P_{00} + u(1-v)P_{10} + (1-u)vP_{01} + uvP_{11}$.

Rewrite as:
$S(u,v) = P_{00} + u(P_{10}-P_{00}) + v(P_{01}-P_{00}) + uv(P_{11} - P_{01} - P_{10} + P_{00})$.
Let $\mathbf{a} = P_{10}-P_{00}$, $\mathbf{b} = P_{01}-P_{00}$, $\mathbf{c} = P_{11} - P_{10} - P_{01} + P_{00}$.

$\frac{\partial S}{\partial u} = \mathbf{a} + v\mathbf{c} = (P_{10}-P_{00}) + v(P_{11}-P_{10}-P_{01}+P_{00})$
$\frac{\partial S}{\partial v} = \mathbf{b} + u\mathbf{c} = (P_{01}-P_{00}) + u(P_{11}-P_{10}-P_{01}+P_{00})$

**Normal Vector:**
$\mathbf{n}(u,v) = \frac{\partial S}{\partial u} \times \frac{\partial S}{\partial v} = (\mathbf{a} + v\mathbf{c}) \times (\mathbf{b} + u\mathbf{c})$
$\mathbf{n}(u,v) = (\mathbf{a} \times \mathbf{b}) + u(\mathbf{a} \times \mathbf{c}) + v(\mathbf{c} \times \mathbf{b}) + uv(\mathbf{c} \times \mathbf{c})$
Since $\mathbf{c} \times \mathbf{c} = 0$:
$$ \mathbf{n}(u,v) = (\mathbf{a} \times \mathbf{b}) + u(\mathbf{a} \times \mathbf{c}) + v(\mathbf{c} \times \mathbf{b}) $$
