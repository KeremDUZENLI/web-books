### Answer 12.5
**Logic:**
1.  **Rotation:** Create Euler rotation matrix $R = R_z(\text{roll}) R_y(\text{yaw}) R_x(\text{pitch})$? (Note: Text usually specifies order, e.g., Yaw-Pitch-Roll or $R_{yz} R_{zx} R_{xy}$).
    * Assuming standard sequence (often Yaw($y$) then Pitch($x$) then Roll($z$)): $R = R_y(\psi) R_x(\theta) R_z(\phi)$.
2.  **Translation:** Create translation matrix $T$ for point $P(x,y,z)$.
3.  **Composition:** $M = T \times R$.

**Resulting Matrix (Column-major logic):**
The upper-left $3 \times 3$ is the rotation $R$. The last column is $[P_x, P_y, P_z, 1]^T$.
