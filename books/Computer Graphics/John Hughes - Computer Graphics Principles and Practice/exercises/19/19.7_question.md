### Exercise 19.7
    
We suggested that one bad way to downsample a $300\times300$ image to a $150\times150$ image was to simply take pixels from every second row and every second column. Suppose the source image is a checkerboard: Pixel (i,j) is white if $i+j$ is even, and it's black if it's odd. At a distance, this image looks uniformly gray.

(a) Show that if we choose pixels from rows and columns with odd indices, the resultant subsampled image is all white.
(b) Show that if we use odd row indices, but even column indices, the resultant subsampled image is all black.
