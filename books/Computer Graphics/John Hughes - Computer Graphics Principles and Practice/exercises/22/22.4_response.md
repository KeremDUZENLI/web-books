### Answer 22.4
    
If we set $P_{-1} = P_0$, the tangent calculation at the start becomes:
$v_0 = \frac{1}{3}(P_1 - P_{-1}) = \frac{1}{3}(P_1 - P_0)$.

In the standard Catmull-Rom formulation (using reflection), the start velocity is $v_0 = \frac{2}{3}(P_1 - P_0)$.
By setting $P_{-1} = P_0$, the magnitude of the starting velocity is cut in half ($1/3$ vs $2/3$). This means the object moving along the spline will start more slowly, accelerating out of the first control point with less momentum than in the standard formulation.
