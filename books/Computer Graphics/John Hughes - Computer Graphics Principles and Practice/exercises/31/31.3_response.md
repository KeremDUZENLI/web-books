### Answer 31.3
    
**Strategy:** Use a deterministic, fixed set of directions (e.g., stratified samples on the hemisphere) for final gathering at every point $P$, rather than generating new random directions at each point.

**Arguments For:**
1.  **Temporal Stability:** In animations, using fixed directions prevents the "shimmering" or "boiling" noise pattern that occurs when random sampling varies from frame to frame on static objects.
2.  **Screen-Space Coherence:** If the directions are fixed (or fixed relative to screen space), adjacent pixels will sample similar parts of the scene. This can improve cache coherence and make denoising filters more effective.
3.  **Optimization:** The directions and their cosine weights can be precomputed once, avoiding runtime trigonometric calculations and random number generation.

**Arguments Against:**
1.  **Structured Artifacts (Aliasing):** If the scene contains small, bright light sources or features that fall between the fixed directions, they will be consistently missed (or hit) at regular intervals. This manifests as banding or Moiré patterns rather than incoherent noise.
2.  **Bias:** Strictly speaking, integration requires random sampling (or very dense quadrature) to be unbiased. A fixed set of $N$ directions is a specific quadrature rule; if the integrand (incident lighting) correlates with the sample pattern (e.g., a grid of lights aligning with sample directions), the result will be systematically incorrect.
3.  **Detail Loss:** Random sampling converts high-frequency detail errors into noise, which the eye ignores or denoisers remove. Fixed sampling converts them into structured patterns or missing features, which are visually objectionable and harder to filter.
