### Answer 14.4
**Algorithm:**
Input: Fan list $F = [v_{center}, v_1, v_2, \dots, v_n]$.
Output: Strip list $S$.

A triangle fan connects the center to every subsequent pair.
Triangles: $(v_c, v_1, v_2), (v_c, v_2, v_3), (v_c, v_3, v_4) \dots$
Triangle strip requires alternating vertices.
Sequence:
1.  Start with $v_1$.
2.  Add $v_{center}$. (Edge $v_1 - v_c$)
3.  Add $v_2$. (Triangle $v_1, v_c, v_2$ - Valid)
4.  To get next triangle $(v_c, v_2, v_3)$ using strip logic:
    * Current strip end: $v_c, v_2$. Next must connect to $v_3$.
    * Next vertex in strip adds triangle $(v_c, v_2, v_{next})$. Set $v_{next} = v_3$.
    * Triangle formed: $(v_2, v_c, v_3)$ (Winding reversed: OK for strip odd index).
5.  Next triangle needed: $(v_c, v_3, v_4)$.
    * Current end: $v_2, v_3$. Next vertex $v_x$ forms $(v_2, v_3, v_x)$.
    * We need $v_c$ involved. But $v_c$ is not adjacent to $v_3$ in the sequence.
    * We must introduce a degenerate triangle to "jump" back to $v_c$.
    * Repeat $v_3$. (Triangle $v_c, v_3, v_3$ - Degenerate/Invisible).
    * Add $v_c$. (Triangle $v_3, v_3, v_c$ - Degenerate).
    * Add $v_4$. (Triangle $v_3, v_c, v_4$ - Valid).

**Output Sequence:**
$v_1, v_c, v_2, v_c, v_3, v_c, v_4, v_c \dots$ (With appropriate repeats/degenerates to swap winding if renderer requires strict culling).
