**Computation:**
Given $Y=1$ for all points.
Coordinates $(x,y)$ range over the CIE chromaticity diagram.
$X = x/y$, $Z = (1-x-y)/y$. (Since $Y=1$).

**$L^*$ Calculation:**
$L^* = 116(1/1)^{1/3} - 16 = 100$.
Thus, all points lie on the plane $L^* = 100$.

**$u^*, v^*$ Calculation:**
$u' = \frac{4X}{X + 15Y + 3Z} = \frac{4(x/y)}{(x/y) + 15 + 3(1-x-y)/y} = \frac{4x}{x + 15y + 3(1-x-y)} = \frac{4x}{-2x + 12y + 3}$.
$v' = \frac{9Y}{X + 15Y + 3Z} = \frac{9y}{-2x + 12y + 3}$.
$u^* = 13(100)(u' - u'_n)$.
$v^* = 13(100)(v' - v'_n)$.

**Plot Description:**
The plot is a projection of the CIE chromaticity diagram onto the $L^*=100$ plane. The shape is a projective transformation of the standard horseshoe, scaled by 1300. The curved spectral locus remains curved, but the spacing is distorted to be more perceptually uniform (expanding the blue region, compressing the green).
