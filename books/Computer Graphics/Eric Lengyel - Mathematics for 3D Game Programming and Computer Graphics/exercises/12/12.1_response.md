### Answer 12.1

1. The plane equation is $x = 10$, which can be written as $x - 10 = 0$. The normal to this plane pointing towards the origin is $N = \langle -1, 0, 0 \rangle$, and $D = 10$. The plane equation is $L = \langle -1, 0, 0, 10 \rangle$.
2. The plane $L'$ shifted by radius $r = 2$ towards the sphere is:
$$L' = \langle N, D - r \rangle = \langle -1, 0, 0, 10 - 2 \rangle = \langle -1, 0, 0, 8 \rangle$$
3. The initial position of the sphere is $P_1 = \langle 0, 0, 0 \rangle$. Using 4D homogeneous coordinates, $P_1 = \langle 0, 0, 0, 1 \rangle$.
4. The velocity is $V = \langle 2, 0, 1 \rangle$. Using 4D coordinates for a direction vector, $V = \langle 2, 0, 1, 0 \rangle$.
5. The collision time $t$ is calculated as:
$$t = -\frac{L' \cdot P_1}{L' \cdot V}$$
$$t = -\frac{\langle -1, 0, 0, 8 \rangle \cdot \langle 0, 0, 0, 1 \rangle}{\langle -1, 0, 0, 8 \rangle \cdot \langle 2, 0, 1, 0 \rangle}$$
$$t = -\frac{0 + 0 + 0 + 8}{-2 + 0 + 0 + 0}$$
$$t = -\frac{8}{-2}$$
$$t = 4 \text{ s}$$
