### Answer 11.4

The knot vector $\{0,0,0,0,1,1,1,1\}$ over 4 control points defines a single cubic Bézier segment. Knot insertion at $t' = 0.5$ splits the curve.

The Böhm subdivision formula for inserting $t'$ into the interval $[t_j, t_{j+1})$ calculates new control points as:
$$P_i' = (1 - \alpha_i) P_{i-1} + \alpha_i P_i$$

Where:
$$\alpha_i = \frac{t' - t_i}{t_{i+3} - t_i}$$

For standard indexing $t_0 \dots t_7$, $t' = 0.5$ falls into $[t_3, t_4) = [0, 1)$, so $k=3$. We calculate $\alpha_i$ for $i = 1, 2, 3$:
$$\alpha_1 = \frac{0.5 - 0}{1 - 0} = 0.5$$
$$\alpha_2 = \frac{0.5 - 0}{1 - 0} = 0.5$$
$$\alpha_3 = \frac{0.5 - 0}{1 - 0} = 0.5$$

Calculating the new control points $P_0'$ through $P_4'$:
$$P_0' = P_0 = \langle 0, 0 \rangle$$
$$P_1' = 0.5 P_0 + 0.5 P_1 = 0.5 \langle 0,0 \rangle + 0.5 \langle 1,2 \rangle = \langle 0.5, 1 \rangle$$
$$P_2' = 0.5 P_1 + 0.5 P_2 = 0.5 \langle 1,2 \rangle + 0.5 \langle 2,2 \rangle = \langle 1.5, 2 \rangle$$
$$P_3' = 0.5 P_2 + 0.5 P_3 = 0.5 \langle 2,2 \rangle + 0.5 \langle 3,0 \rangle = \langle 2.5, 1 \rangle$$
$$P_4' = P_3 = \langle 3, 0 \rangle$$
