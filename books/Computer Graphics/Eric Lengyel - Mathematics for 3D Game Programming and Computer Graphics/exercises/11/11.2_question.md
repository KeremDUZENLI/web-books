### Exercise 11.2

Suppose that the de Casteljau algorithm is used to split a Bézier curve having the control points $P_0, P_1, P_2,$ and $P_3$ at the parameter value $t = 1/2$. Find the matrix $M_Q$ that transforms the control points of the original curve into the control points $Q_0, Q_1, Q_2,$ and $Q_3$ for the curve coinciding with the interval $[0, 1/2]$, and find the matrix $M_R$ that transforms the control points of the original curve into the control points $R_0, R_1, R_2,$ and $R_3$ for the curve coinciding with the interval $[1/2, 1]$. That is, find matrices $M_Q$ and $M_R$ such that
$$[Q_0 \ Q_1 \ Q_2 \ Q_3] = [P_0 \ P_1 \ P_2 \ P_3] M_Q$$
and
$$[R_0 \ R_1 \ R_2 \ R_3] = [P_0 \ P_1 \ P_2 \ P_3] M_R$$
