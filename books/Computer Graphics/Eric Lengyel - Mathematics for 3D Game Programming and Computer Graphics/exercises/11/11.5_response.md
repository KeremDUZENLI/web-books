### Answer 11.5

1. Derivatives of $P(t)$:
$$P(t) = \langle r \cos t, r \sin t, ct \rangle$$
$$P'(t) = \langle -r \sin t, r \cos t, c \rangle$$
$$P''(t) = \langle -r \cos t, -r \sin t, 0 \rangle$$
$$P'''(t) = \langle r \sin t, -r \cos t, 0 \rangle$$

2. Magnitude of the first derivative:
$$||P'(t)|| = \sqrt{(-r \sin t)^2 + (r \cos t)^2 + c^2} = \sqrt{r^2 + c^2}$$

3. Cross product $P'(t) \times P''(t)$:
$$P'(t) \times P''(t) = \langle c r \sin t, -c r \cos t, r^2 \sin^2 t + r^2 \cos^2 t \rangle = \langle c r \sin t, -c r \cos t, r^2 \rangle$$

4. Magnitude of the cross product:
$$||P'(t) \times P''(t)|| = \sqrt{c^2 r^2 \sin^2 t + c^2 r^2 \cos^2 t + r^4} = \sqrt{c^2 r^2 + r^4} = r \sqrt{r^2 + c^2}$$

5. Calculate Curvature $\kappa(t)$:
$$\kappa(t) = \frac{||P'(t) \times P''(t)||}{||P'(t)||^3} = \frac{r \sqrt{r^2 + c^2}}{(\sqrt{r^2 + c^2})^3} = \frac{r}{r^2 + c^2}$$

6. Dot product $(P'(t) \times P''(t)) \cdot P'''(t)$:
$$\langle c r \sin t, -c r \cos t, r^2 \rangle \cdot \langle r \sin t, -r \cos t, 0 \rangle = c r^2 \sin^2 t + c r^2 \cos^2 t + 0 = c r^2$$

7. Calculate Torsion $\tau(t)$:
$$\tau(t) = \frac{(P'(t) \times P''(t)) \cdot P'''(t)}{||P'(t) \times P''(t)||^2} = \frac{c r^2}{(r \sqrt{r^2 + c^2})^2} = \frac{c r^2}{r^2(r^2 + c^2)} = \frac{c}{r^2 + c^2}$$
