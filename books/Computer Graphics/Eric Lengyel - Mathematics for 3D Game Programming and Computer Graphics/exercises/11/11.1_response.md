### Answer 11.1

Multiply $B_2(t)$ by $((1-t) + t)$, which equals 1:
$$B_2(t) = ((1 - t) + t) \left( (1 - t)^2 P_0 + 2t(1 - t)P_1 + t^2 P_2 \right)$$

Distribute the terms:
$$B_2(t) = (1 - t)^3 P_0 + 2t(1 - t)^2 P_1 + t^2(1 - t) P_2 + t(1 - t)^2 P_0 + 2t^2(1 - t) P_1 + t^3 P_2$$

Group by powers of $t$ and $(1-t)$:
$$B_2(t) = (1 - t)^3 P_0 + t(1 - t)^2 (P_0 + 2P_1) + t^2(1 - t) (2P_1 + P_2) + t^3 P_2$$

Equate this to the cubic Bézier formula $B_3(t) = (1 - t)^3 P_0' + 3t(1 - t)^2 P_1' + 3t^2(1 - t) P_2' + t^3 P_3'$:

- $P_0' = P_0$
- $3P_1' = P_0 + 2P_1 \implies P_1' = \frac{1}{3}P_0 + \frac{2}{3}P_1$
- $3P_2' = 2P_1 + P_2 \implies P_2' = \frac{2}{3}P_1 + \frac{1}{3}P_2$
- $P_3' = P_2$
