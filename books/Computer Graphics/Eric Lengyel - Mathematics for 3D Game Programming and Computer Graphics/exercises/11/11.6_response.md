### Answer 11.6

Let the path be parameterized by arc length $s$. We use the Frenet-Serret formulas: 
- Unit Tangent: $T = \frac{dP}{ds}$
- Principal Normal relation: $\frac{dT}{ds} = \kappa N$
- Binormal relation: $\frac{dN}{ds} = -\kappa T + \tau B$
- Binormal definition: $B = T \times N$

1. Express derivatives with respect to arc length $s$:
$$\frac{dP}{ds} = T$$
$$\frac{d^2P}{ds^2} = \frac{dT}{ds} = \kappa N$$
$$\frac{d^3P}{ds^3} = \frac{d}{ds}(\kappa N) = \kappa' N + \kappa \frac{dN}{ds} = \kappa' N + \kappa(-\kappa T + \tau B) = -\kappa^2 T + \kappa' N + \kappa \tau B$$

2. Substitute into the cross product $(\frac{d^2P}{ds^2} \times \frac{d^3P}{ds^3})$:
$$(\kappa N) \times (-\kappa^2 T + \kappa' N + \kappa \tau B)$$
$$= -\kappa^3 (N \times T) + \kappa \kappa' (N \times N) + \kappa^2 \tau (N \times B)$$

3. Apply cross product identities ($N \times N = 0$, $N \times T = -B$, $N \times B = T$):
$$= -\kappa^3 (-B) + 0 + \kappa^2 \tau (T)$$
$$= \kappa^3 B + \kappa^2 \tau T$$

4. Substitute into the dot product $\frac{dP}{ds} \cdot (\frac{d^2P}{ds^2} \times \frac{d^3P}{ds^3})$:
$$T \cdot (\kappa^3 B + \kappa^2 \tau T)$$

5. Distribute the dot product ($T \cdot B = 0$ and $T \cdot T = 1$):
$$= \kappa^3 (T \cdot B) + \kappa^2 \tau (T \cdot T)$$
$$= \kappa^3 (0) + \kappa^2 \tau (1)$$
$$= \kappa^2 \tau$$

Q.E.D.
