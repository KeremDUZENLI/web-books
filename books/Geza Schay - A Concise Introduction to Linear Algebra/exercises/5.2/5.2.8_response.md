### Answer 5.2.8

**Vectors:** $\mathbf{a}_1=(0,0,-1,1), \mathbf{a}_2=(1,0,0,1), \mathbf{a}_3=(1,0,-1,0)$.

**1. Orthonormal Basis (Gram-Schmidt):**
* $\mathbf{u}_1 = \mathbf{a}_1$. $\|\mathbf{u}_1\| = \sqrt{2}$. $\mathbf{q}_1 = (0,0,-1,1)/\sqrt{2}$.
* $\mathbf{u}_2 = \mathbf{a}_2 - \text{proj}_{\mathbf{u}_1}\mathbf{a}_2$.
    $\mathbf{a}_2 \cdot \mathbf{u}_1 = 1$. $\|\mathbf{u}_1\|^2 = 2$.
    $\mathbf{u}_2 = (1,0,0,1) - 0.5(0,0,-1,1) = (1, 0, 0.5, 0.5)$.
    Scale to $(2, 0, 1, 1)$. $\|\mathbf{u}'_2\|^2 = 6$.
    $\mathbf{q}_2 = (2,0,1,1)/\sqrt{6}$.
* $\mathbf{u}_3 = \mathbf{a}_3 - \text{proj}_{\mathbf{q}_1}\mathbf{a}_3 - \text{proj}_{\mathbf{q}_2}\mathbf{a}_3$.
    $\mathbf{a}_3 \cdot \mathbf{u}_1 = 1$. Proj 1 = $0.5(0,0,-1,1) = (0,0,-0.5,0.5)$.
    $\mathbf{a}_3 \cdot \mathbf{u}'_2 = 2 - 1 = 1$. Proj 2 = $\frac{1}{6}(2,0,1,1)$.
    $\mathbf{u}_3 = (1,0,-1,0) - (0,0,-0.5,0.5) - (1/3, 0, 1/6, 1/6)$.
    $= (1, 0, -0.5, -0.5) - (0.33, 0, 0.16, 0.16) = (0.66, 0, -0.66, -0.66)$.
    Scale to $(1, 0, -1, -1)$.
    $\mathbf{q}_3 = (1, 0, -1, -1)/\sqrt{3}$.

**2. Extend Basis:**
We need a vector orthogonal to all (e.g., in Nullspace of matrix with rows $\mathbf{a}_i$).
Notice the 2nd component is always 0.
$\mathbf{q}_4 = (0, 1, 0, 0)^T$ is orthogonal to all of them.

$R_{11} = \|\mathbf{a}_1\| = \sqrt{2}$.
$R_{12} = \mathbf{q}_1^T \mathbf{a}_2 = 1/\sqrt{2}$.
$R_{22} = \|\mathbf{u}_2\| = \sqrt{6}/2$.
