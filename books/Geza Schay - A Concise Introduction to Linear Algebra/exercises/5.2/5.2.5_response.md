### Answer 5.2.5

**Given:** $Q^{-1} = Q^T$.
This implies $Q^T Q = I$ and $Q Q^T = I$.

**Definition of Orthogonal:**
A real square matrix is orthogonal if its columns (and rows) form an orthonormal set.
Check columns $\mathbf{q}_i$:
The $(i, j)$ entry of $Q^T Q$ is $\mathbf{q}_i^T \mathbf{q}_j$.
Since $Q^T Q = I$, $\mathbf{q}_i^T \mathbf{q}_j = \delta_{ij}$ (1 if $i=j$, 0 else).
This means the columns are orthonormal.
Thus, $Q$ is an orthogonal matrix.
