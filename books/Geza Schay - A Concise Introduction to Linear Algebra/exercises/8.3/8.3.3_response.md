### Answer 8.3.3

**Proof:**
$B = A - cI$.
The eigenvalues of $B$ are $\mu_i = \lambda_i - c$, where $\lambda_i$ are eigenvalues of $A$.
$B$ is singular if and only if one of its eigenvalues is 0.
$\mu_i = 0 \iff \lambda_i - c = 0 \iff c = \lambda_i$.
If $c$ is **not** an eigenvalue of $A$, then $\lambda_i - c \neq 0$ for all $i$.
Thus no eigenvalue of $B$ is 0.
Therefore, $B$ is nonsingular.
