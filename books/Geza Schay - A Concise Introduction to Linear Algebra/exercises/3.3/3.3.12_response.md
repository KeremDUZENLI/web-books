### Answer 3.3.12

**Given:** $\{\mathbf{a}_1, \mathbf{a}_2\}$ independent. $\mathbf{v} \notin \text{Span}\{\mathbf{a}_1, \mathbf{a}_2\}$.

**Proof (by Contradiction):**
Assume $\{\mathbf{v}, \mathbf{a}_1, \mathbf{a}_2\}$ is dependent.
Then there exist scalars $c_1, c_2, c_3$ not all zero such that $c_1\mathbf{v} + c_2\mathbf{a}_1 + c_3\mathbf{a}_2 = \mathbf{0}$.
Case 1: $c_1 \neq 0$.
Then $\mathbf{v} = -\frac{c_2}{c_1}\mathbf{a}_1 - \frac{c_3}{c_1}\mathbf{a}_2$.
This implies $\mathbf{v} \in \text{Span}\{\mathbf{a}_1, \mathbf{a}_2\}$, which contradicts the given assumption.
Case 2: $c_1 = 0$.
Then $c_2\mathbf{a}_1 + c_3\mathbf{a}_2 = \mathbf{0}$ with $c_2, c_3$ not both zero (since at least one $c$ was nonzero).
This implies $\{\mathbf{a}_1, \mathbf{a}_2\}$ is dependent, which contradicts the given assumption.

Since both cases lead to a contradiction, the set must be **Independent**.
