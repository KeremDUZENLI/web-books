### Answer 3.3.19

**Proof:**
Vectors $\mathbf{a}_1 \dots \mathbf{a}_m$ in $\mathbb{R}^n$. Matrix $A$ is $n \times m$.
**Independent $\iff$ Rank $m$.**
1.  If Independent: No non-trivial relation $\implies A\mathbf{x}=\mathbf{0}$ has only $\mathbf{x}=\mathbf{0}$.
    This means there are no free variables. Every column must be a pivot column.
    Number of pivots = $m$. Rank = $m$.
    (Note: This requires $m \le n$ because rank cannot exceed $n$).
2.  If Rank $m$: There are $m$ pivots. Since there are $m$ columns, every column is a pivot column.
    No free variables $\implies$ trivial solution only $\implies$ Independent.
