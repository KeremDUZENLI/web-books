### Answer 3.3.11

**Vectors:**
$\mathbf{a}_1=(1,0,0,1)$
$\mathbf{a}_2=(0,0,1,1)$
$\mathbf{a}_3=(1,1,0,0)$
$\mathbf{a}_4=(1,0,1,1)$

**Statement:** **Dependent.**

**Proof:**
Look at $\mathbf{a}_4$.
$\mathbf{a}_4 = (1, 0, 1, 1)$.
Compare to $\mathbf{a}_1 + \mathbf{a}_2 = (1, 0, 0, 1) + (0, 0, 1, 1) = (1, 0, 1, 2)$. Close but not quite.
Let's solve $c_1\mathbf{a}_1 + c_2\mathbf{a}_2 + c_3\mathbf{a}_3 + c_4\mathbf{a}_4 = \mathbf{0}$.
Row reduction of matrix:
$$ \begin{bmatrix} 1 & 0 & 1 & 1 \\ 0 & 0 & 1 & 0 \\ 0 & 1 & 0 & 1 \\ 1 & 1 & 0 & 1 \end{bmatrix} $$
$R_4 - R_1 \implies (0, 1, -1, 0)$.
Swap rows to echelon:
1. $R_1$: $(1, 0, 1, 1)$.
2. $R_3 \to R_2$: $(0, 1, 0, 1)$.
3. $R_2 \to R_3$: $(0, 0, 1, 0)$.
4. $R_4$ (modified) was $(0, 1, -1, 0)$.
   $R_4 - R_2 \implies (0, 0, -1, -1)$.
   $R_4 + R_3 \implies (0, 0, 0, -1)$.
   Pivots in all 4 columns.
   Wait, calculation check.
   $R_4$ init: $1,1,0,1$.
   $R_4 - R_1$: $0, 1, -1, 0$.
   Current: $R_2$ is $(0,1,0,1)$.
   $R_4 - R_2$: $(0, 0, -1, -1)$.
   $R_3$ is $(0,0,1,0)$.
   $R_4 + R_3$: $(0, 0, 0, -1)$.
   Four pivots. The vectors are **Independent**.
