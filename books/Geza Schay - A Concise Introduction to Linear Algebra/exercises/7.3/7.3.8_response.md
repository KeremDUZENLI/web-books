### Answer 7.3.8

**a. Polar Plot Logic:**
The code defines an angle $t$.
It calculates radius $ra$ for an ellipse equation derived from polar form $x = r \cos t, y = r \sin t$.
Substituting into $ax^2 + bxy + cy^2 = 1 \implies r^2(a c^2 + b cs + c s^2) = 1 \implies r = 1/\sqrt{\dots}$.
This plots the rotated ellipse.
$rb$ plots the standard ellipse (no $xy$ term).

**b. Matrix Method:**
`x = [cos(t); sin(t)]` creates a list of unit vectors.
`x'*A*x` computes the quadratic form value $v = \mathbf{u}^T A \mathbf{u}$ for each unit direction.
Since the equation is $\mathbf{r}^T A \mathbf{r} = 1$ and $\mathbf{r} = \rho \mathbf{u}$, we have $\rho^2 (\mathbf{u}^T A \mathbf{u}) = 1$.
So $\rho = 1/\sqrt{v}$.
`polar(t, r)` plots this radius against the angle.
