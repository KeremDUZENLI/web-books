### Answer 7.2.2

**Definition:**
For a diagonalizable matrix $A = S \Lambda S^{-1}$ where $\Lambda = \text{diag}(\lambda_1, \dots, \lambda_n)$ and all $\lambda_i > 0$, we define $A^x$ for any real number $x$ as:
$$ A^x = S \Lambda^x S^{-1} $$
where $\Lambda^x = \text{diag}(\lambda_1^x, \lambda_2^x, \dots, \lambda_n^x)$.

This definition is consistent because the power function $f(t) = t^x$ is well-defined for positive real numbers.
