### Answer 3.6.18

**Equivalence Relation:**
1.  **Reflexive:** $A = I^{-1} A I$. $A \sim A$.
2.  **Symmetric:** $A \sim B \implies A = S^{-1} B S$.
    Multiply by $S$ on left, $S^{-1}$ on right: $S A S^{-1} = B$.
    Let $Q = S^{-1}$. $B = Q^{-1} A Q$. So $B \sim A$.
3.  **Transitive:** $A \sim B$ ($A = S^{-1} B S$) and $B \sim C$ ($B = T^{-1} C T$).
    $A = S^{-1} (T^{-1} C T) S = (TS)^{-1} C (TS)$.
    $A \sim C$.
