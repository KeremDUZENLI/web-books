### Answer 1.3.16

**Given:** $P_0(1, -2, 4)$ parallel to plane $3x - 4y + 3z = 7$.

1.  **Find Normal:**
    The parallel plane has normal $\mathbf{n} = (3, -4, 3)$. Our new plane uses the same normal.

2.  **Nonparametric Form:**
    $3(x-1) - 4(y+2) + 3(z-4) = 0$.
    $3x - 4y + 3z - 3 - 8 - 12 = 0$.
    $3x - 4y + 3z = 23$.

3.  **Parametric Form:**
    Find two vectors orthogonal to $(3, -4, 3)$.
    * $\mathbf{u} = (4, 3, 0)$.
    * $\mathbf{v} = (1, 0, -1)$.
    $\mathbf{p} = (1, -2, 4) + s(4, 3, 0) + t(1, 0, -1)$.
