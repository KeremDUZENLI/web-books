### Answer 1.3.37

**Given:**
$L_1: (2, 1, 5) + s(-4, 0, 3)$ (Direction $\mathbf{v}_1$)
$L_2: (0, -2, 3) + t(5, 0, -2)$ (Direction $\mathbf{v}_2$)

1.  **Direction of Transverse ($L$):**
    $\mathbf{v}_L = \mathbf{v}_1 \times \mathbf{v}_2$.
    Since both have $y=0$, cross product is pure $y$: $(0, 1, 0)$ (normalized).
    So $L$ is a vertical line.

2.  **Plane $S$:**
    Contains $L_1$ and $\mathbf{v}_L = (0, 1, 0)$.
    Normal $\mathbf{n}_S = \mathbf{v}_1 \times \mathbf{v}_L = (-4, 0, 3) \times (0, 1, 0) = (-3, 0, -4)$.
    Passes through $(2, 1, 5)$.
    Eq: $-3(x-2) - 4(z-5) = 0 \implies 3x + 4z = 26$.

3.  **Intersection of $S$ and $L_2$:**
    $3(5t) + 4(3-2t) = 26$.
    $15t + 12 - 8t = 26$.
    $7t = 14 \implies t = 2$.
    Point on $L_2$: $(10, -2, -1)$.

4.  **Equation of $L$:**
    Through $(10, -2, -1)$ with direction $(0, 1, 0)$.
    $\mathbf{p} = (10, -2, -1) + u(0, 1, 0)$.
