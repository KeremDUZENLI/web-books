### Answer 3.1.8

**No, this is not a vector space.**

Definition: $(p_1, p_2) + (q_1, q_2) = (p_1 + q_2, 0)$.
Let's check the **Identity Element (Zero Vector)**.
We need $\mathbf{p} + \mathbf{0} = \mathbf{p}$.
$(p_1, p_2) + (z_1, z_2) = (p_1 + z_2, 0)$.
We need this result to equal $(p_1, p_2)$.
So $(p_1 + z_2, 0) = (p_1, p_2)$.
This implies $p_2 = 0$.
But $p_2$ can be any real number. If we pick $\mathbf{p} = (1, 5)$, the sum results in $(1+z_2, 0)$, which can never equal $(1, 5)$.
Thus, there is **no zero vector**.
