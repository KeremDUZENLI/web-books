### Answer 4.1.4

**a. $T(\mathbf{x}) = \mathbf{a} \cdot \mathbf{x}$:**
Linear. (Dot product is linear in each argument). Matrix is row vector $\mathbf{a}^T$.

**b. $T(\mathbf{x}) = (A - \lambda I)\mathbf{x}$:**
Linear. Matrix multiplication is linear. Matrix is $M = A - \lambda I$.

**c. $T(\mathbf{x}) = A\mathbf{x} + \mathbf{b}$ ($\mathbf{b} \neq 0$):**
Not Linear. $T(\mathbf{0}) = \mathbf{b} \neq \mathbf{0}$. (This is Affine).

**d. $T(\mathbf{x}) = |\mathbf{x}|$:**
Not Linear. $|c\mathbf{x}| = |c| |\mathbf{x}| \neq c |\mathbf{x}|$ for $c < 0$. Also $|\mathbf{x}+\mathbf{y}| \neq |\mathbf{x}|+|\mathbf{y}|$ usually.

**e. $T(\mathbf{x}) = \mathbf{x}^T \mathbf{a}$:**
Linear. $\mathbf{x}^T \mathbf{a} = \mathbf{x} \cdot \mathbf{a}$. Same as (a).

**f. $T(\mathbf{x}) = (\mathbf{a}^T \mathbf{x})\mathbf{b}$:**
Linear. The scalar $(\mathbf{a}^T \mathbf{x})$ scales the vector $\mathbf{b}$.
$T(c\mathbf{x}) = (\mathbf{a}^T c\mathbf{x})\mathbf{b} = c(\mathbf{a}^T \mathbf{x})\mathbf{b} = cT(\mathbf{x})$.
Additivity holds similarly. Matrix is $\mathbf{b}\mathbf{a}^T$ (Outer product).
