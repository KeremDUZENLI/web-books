### Answer 4.1.22

**a. Bases:**
$B$ from `rref(A')` (rows). $C$ from `rref(A)` (cols).

**b. Matrix $[T]$ (Standard):**
We need $T$ s.t. $T(\mathbf{b}_i) = \mathbf{c}_i$.
Basis matrices $M_B$ (cols are $b_i$) and $M_C$ (cols are $c_i$).
$[T] = M_C M_B^{-1}$ (if $M_B$ square).
However, $Row(A)$ and $Col(A)$ are subspaces of $\mathbb{R}^4$, usually dim < 4.
If dim < 4, $M_B$ is not square invertible.
$T$ is defined on $Row(A)$. We can extend to $\mathbb{R}^4$ (e.g. 0 on null space).
Using pseudoinverse: $[T] = M_C M_B^+$.

**c. Matrix $T_{B,C}$:**
Relative to basis $B$ input and $C$ output.
$T(\mathbf{b}_1) = \mathbf{c}_1 = 1 \cdot \mathbf{c}_1 + 0 \dots$
$T(\mathbf{b}_i) = \mathbf{c}_i$.
The matrix is simply the Identity matrix $I_r$ ($r \times r$).
