### Answer 4.1.3

**Statement:** Linear $\iff$ Preserves Lines?
**Correction:** Not exactly. Affine transformations ($T(\mathbf{x}) = A\mathbf{x} + \mathbf{b}$) also preserve lines but are not linear if $\mathbf{b} \neq \mathbf{0}$ (doesn't map $\mathbf{0}$ to $\mathbf{0}$).
**Theorem:** A mapping $T$ is linear if and only if it preserves lines passing through the origin (lines $\mathbf{x} = t\mathbf{a}$ map to lines through origin) AND preserves the "grid" structure (additivity).
Actually, the property "preserves straight lines" usually defines **Affine** transformations.
To be **Linear**, it must preserve lines AND fix the origin ($T(\mathbf{0}) = \mathbf{0}$).
**Proof:**
If Linear: Line $\mathbf{x}_0 + t\mathbf{a} \to T(\mathbf{x}_0) + tT(\mathbf{a})$. This is a line (or point if $T(\mathbf{a})=\mathbf{0}$).
If Line-Preserving + Origin-Fixing:
Maps line through $\mathbf{0}$ to line through $\mathbf{0} \implies T(t\mathbf{x}) = tT(\mathbf{x})$.
Maps parallelogram grid to parallelogram grid $\implies T(\mathbf{x}+\mathbf{y}) = T(\mathbf{x}) + T(\mathbf{y})$.
Thus Linear.
