### Answer 7.4.2

**Vectors:** $\mathbf{x}=\begin{bmatrix}2+4i\\ 1-2i\end{bmatrix}, \quad \mathbf{y}=\begin{bmatrix}1-5i\\ 4+2i\end{bmatrix}$.

**a. Conjugates:**
* $\mathbf{x}^H = [2-4i, 1+2i]$.
* $\mathbf{y}^H = [1+5i, 4-2i]$.

**b. Lengths:**
* $|\mathbf{x}|^2 = |2+4i|^2 + |1-2i|^2 = (4+16) + (1+4) = 25 \implies |\mathbf{x}| = 5$.
* $|\mathbf{y}|^2 = |1-5i|^2 + |4+2i|^2 = (1+25) + (16+4) = 46 \implies |\mathbf{y}| = \sqrt{46}$.

**c. Inner Products:**
* $\mathbf{x}^H \mathbf{y} = (2-4i)(1-5i) + (1+2i)(4+2i) = (2 - 10i - 4i - 20) + (4 + 2i + 8i - 4) = (-18 - 14i) + (10i) = -18 - 4i$.
* $\mathbf{y}^H \mathbf{x} = \overline{-18 - 4i} = -18 + 4i$.
