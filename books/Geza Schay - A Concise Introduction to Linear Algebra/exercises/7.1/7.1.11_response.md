### Answer 7.1.11

**Proof:**
$\det(A) = \text{product of eigenvalues} = \prod \lambda_i$.
A matrix is singular if and only if $\det(A) = 0$.
$\det(A) = 0 \iff \prod \lambda_i = 0 \iff$ at least one $\lambda_i = 0$.
Alternatively, if $\lambda = 0$ is an eigenvalue, there exists $\mathbf{x} \neq \mathbf{0}$ such that $A\mathbf{x} = 0\mathbf{x} = \mathbf{0}$. This means $Null(A)$ is non-trivial, so $A$ is singular.
