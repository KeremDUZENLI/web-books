### Answer 2.3.7

**System:**
1. $2x_1 + 3x_2 - x_3 = 4$
2. $3x_1 + 5x_2 + 2x_3 = 1$

**RREF:**
1.  $R_1 \leftarrow R_1/2 \implies (1, 1.5, -0.5 \mid 2)$.
2.  $R_2 \leftarrow R_2 - 3R_1 \implies 1 - 3(2) = -5$. Row: $(0, 0.5, 3.5 \mid -5)$.
3.  $R_2 \leftarrow 2R_2 \implies (0, 1, 7 \mid -10)$.
4.  $R_1 \leftarrow R_1 - 1.5R_2 \implies 2 - 1.5(-10) = 17$. Row: $(1, 0, -11 \mid 17)$.
    $$ \left[ \begin{array}{ccc|c} 1 & 0 & -11 & 17 \\ 0 & 1 & 7 & -10 \end{array} \right] $$

**General Solution:**
$x_3 = t$.
$x_2 = -10 - 7t$.
$x_1 = 17 + 11t$.
Vector form: $\mathbf{x} = (17, -10, 0) + t(11, -7, 1)$.

**Components:**
* Homogeneous solution $\mathbf{v} = t(11, -7, 1)$.
* Particular solution $\mathbf{x}_b$ (let $t=0$): $(17, -10, 0)$.
* Particular solution $\mathbf{x}'_b$ (let $t=1$): $(28, -17, 1)$.

**Equivalence:**
Show $\{\mathbf{x}_b + \mathbf{v}\}$ and $\{\mathbf{x}'_b + \mathbf{v}\}$ are the same set.
Any vector in the second set looks like: $(28, -17, 1) + s(11, -7, 1)$.
$= (17, -10, 0) + (11, -7, 1) + s(11, -7, 1)$
$= (17, -10, 0) + (1+s)(11, -7, 1)$.
This is just the first set with parameter $t = 1+s$. Since $s$ ranges over all reals, $t$ ranges over all reals. The lines are identical.
