# Chapter 22: Splines and Subdivision Curves

---

## Main Idea

- This chapter introduces the mathematical foundations of creating smooth curves from discrete data points using splines and subdivision techniques.
- It details specific formulations including Hermite, Bézier, Catmull-Rom, and B-splines, focusing on how different basis matrices and geometry matrices control curve properties.

---

## Keywords

**Spline:**

- A piecewise polynomial curve used to interpolate or approximate a sequence of control points.

**Hermite Curve:**

- A cubic polynomial curve segment defined by its start and end points and the tangent vectors at those points.

**Bézier Curve:**

- A parametric curve defined by a set of control points where the curve is contained within the convex hull of the points and interpolates the endpoints.

**Catmull-Rom Spline:**

- An interpolating spline where the tangent at each control point is calculated based on the positions of the adjacent control points.

**B-spline:**

- A spline curve defined by control points that typically does not pass through the points but offers higher continuity and local control.

**Subdivision Curve:**

- A smooth limit curve generated by recursively refining a control polygon through splitting and averaging rules.

**Geometry Matrix ($G$):**

- A matrix containing the control data (points or vectors) used to define a specific curve segment.

**Basis Matrix ($M$):**

- A matrix of coefficients that blends the geometry data with the power basis vectors to define the shape of the curve.

## Formulas

**General Matrix Form for Cubic Curves**

- **Formula:** $\gamma(t) = G M T(t)$

- **Meaning:** Defines any cubic curve segment as the product of geometry $G$, basis matrix $M$, and time vector $T(t) = [1, t, t^2, t^3]^T$.

- **Use in graphics:** Standardizing the evaluation and transformation of different curve types in software.

**Hermite Curve Equation**

- **Formula:** $\gamma(t) = (2t^3 - 3t^2 + 1)P + (-2t^3 + 3t^2)Q + (t^3 - 2t^2 + t)v + (t^3 - t^2)w$

- **Meaning:** Explicit polynomial form weighting the endpoints $P, Q$ and tangents $v, w$.

- **Use in graphics:** Interpolating motion between keyframes with specified velocities.

**Catmull-Rom Tangent Formula**

- **Formula:** $v_i = \frac{1}{3}(P_{i+1} - P_{i-1})$

- **Meaning:** Calculates the tangent at control point $i$ using the vector between its neighbors.

- **Use in graphics:** Generating smooth paths through a set of points without manual tangent specification.

## Practical Use

**Keyframe Interpolation**

- Using Catmull-Rom splines to calculate the position of an animated object at intermediate frames so it moves smoothly through defined waypoints.

**Vector Graphics**

- Using Bézier curves to define scalable fonts and shapes that remain smooth at any resolution.
