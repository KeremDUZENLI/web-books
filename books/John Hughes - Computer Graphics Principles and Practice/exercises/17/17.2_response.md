### Answer 17.2

**Run-Length Encoding (RLE):**
For images with large regions of constant color, **Run-Length Encoding** is an effective lossless compression method.
- Instead of storing every pixel value individually (e.g., "Red, Red, Red, Red..."), store the value and the count of consecutive occurrences.
- Format: `(Value, Count)`
- Example: `(Red, 50), (Blue, 20)` represents 50 red pixels followed by 20 blue pixels.
- This dramatically reduces file size for uniform areas (like diagrams or cartoons) while preserving exact data.
