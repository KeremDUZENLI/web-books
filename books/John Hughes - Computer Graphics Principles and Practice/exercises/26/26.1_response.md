### Answer 26.1
    
The projection map $p$ from the cylinder $x^2 + z^2 = 1$ to the unit sphere is defined by:
$p(x, y, z) = (\frac{x}{\sqrt{x^2 + z^2}}, y, \frac{z}{\sqrt{x^2 + z^2}})$
Note: This formulation from the text assumes $x, z$ are coordinates on the cylinder surface. On the cylinder $x^2 + z^2 = 1$, so the map simplifies to $p(x, y, z) = (x, y, z)$?
Actually, the text (Equation 26.22) defines the map $p: C \to S$ where $C$ is a cylinder of radius 1. If $(x,z)$ are on the cylinder, $\sqrt{x^2+z^2}=1$. The formula likely implies projecting from a general point or parameterization.
Let's parameterize the cylinder using $(\theta, y)$. A point on the cylinder is $(\cos\theta, y, \sin\theta)$.
The map $p$ projects horizontally to the sphere. The corresponding point on the sphere is determined by the same angle $\theta$ and height $y$.
However, on the sphere $x_s^2 + y_s^2 + z_s^2 = 1$, so the radius at height $y$ is $\sqrt{1-y^2}$.
The map described in the text $p(x, y, z)$ seems to project *from* the cylinder *to* the sphere?
Standard Archimedes projection: A point $(\cos\theta, y, \sin\theta)$ on the cylinder maps to $(\sqrt{1-y^2}\cos\theta, y, \sqrt{1-y^2}\sin\theta)$ on the sphere.

**Jacobian Computation at $z=0$ (or $\theta=0$):**
Cylinder parameters: $u = \theta, v = y$.
Point on cylinder: $C(u,v) = (\cos u, v, \sin u)$.
Tangent vectors: $C_u = (-\sin u, 0, \cos u)$, $C_v = (0, 1, 0)$.
Area element $dA_{cyl} = |C_u \times C_v| du dv = 1 du dv$.

Map to sphere $S(u,v) = (\sqrt{1-v^2}\cos u, v, \sqrt{1-v^2}\sin u)$.
Tangent vectors:
$S_u = (-\sqrt{1-v^2}\sin u, 0, \sqrt{1-v^2}\cos u)$
$S_v = (\frac{-v}{\sqrt{1-v^2}}\cos u, 1, \frac{-v}{\sqrt{1-v^2}}\sin u)$

At $(x,y,0)$ on the cylinder, $\sin u = 0, \cos u = 1$ (assuming $z=0$ implies alignment with x-axis).
$S_u = (0, 0, \sqrt{1-v^2})$
$S_v = (\frac{-v}{\sqrt{1-v^2}}, 1, 0)$
Cross product $S_u \times S_v = (-\sqrt{1-v^2}, \frac{-v}{\sqrt{1-v^2}} \cdot \sqrt{1-v^2}, 0) = (-\sqrt{1-v^2}, -v, 0)$? No.
Let's compute cross product determinant:
$\mathbf{i}(0 - \sqrt{1-v^2}) - \mathbf{j}(0 - 0) + \mathbf{k}(0 - 0)$?
Wait, $S_u = (0, 0, A)$ where $A = \sqrt{1-v^2}$.
$S_v = (B, 1, 0)$.
$S_u \times S_v = \det \begin{vmatrix} \mathbf{i} & \mathbf{j} & \mathbf{k} \\ 0 & 0 & A \\ B & 1 & 0 \end{vmatrix} = \mathbf{i}(-A) - \mathbf{j}(-AB) + \mathbf{k}(0) = (-A, AB, 0)$.
Magnitude $|S_u \times S_v| = \sqrt{A^2 + A^2 B^2} = A \sqrt{1 + B^2}$.
Substitute $A = \sqrt{1-v^2}$ and $B = \frac{-v}{\sqrt{1-v^2}}$.
$|S_u \times S_v| = \sqrt{1-v^2} \sqrt{1 + \frac{v^2}{1-v^2}} = \sqrt{1-v^2} \sqrt{\frac{1-v^2+v^2}{1-v^2}} = \sqrt{1-v^2} \frac{1}{\sqrt{1-v^2}} = 1$.

Since the area element on the sphere is $1 du dv$, which matches the area element on the cylinder, the map is area-preserving.
**Sufficiency of $z=0$:** The cylinder and sphere are rotationally symmetric about the y-axis. The local stretching properties depend only on $y$, not on the angle $\theta$ (or position x, z). Therefore, verifying at one angle ($z=0$) is sufficient.
