### Answer 38.3
    
**Comparison:**
* **1-to-1 Mapping:** Best cache coherence. Adjacent pixels map to adjacent texels. Performance should be highest.
* **Minification (many texels per pixel):** If the texture is not mipmapped, the cache miss rate increases significantly because adjacent pixels access widely separated texels (striding). Performance drops.
* **Magnification (many pixels per texel):** High coherence (many pixels hit the same texel line). Performance remains high, similar to or better than 1-to-1.

**Estimating Bandwidth:**
If the shader does minimal math (just a texture fetch), the system is bandwidth bound.
Bandwidth $B \approx \frac{\text{Total Bytes Transferred}}{\text{Time}}$.
For a $256 \times 256$ triangle (approx $32k$ pixels) using 4-byte texels:
If using multiple textures forces a unique fetch per pixel per texture:
$\text{Bytes} = (\text{Pixels}) \times (\text{Textures}) \times (4 \text{ bytes})$.
If frame rate is $F$ frames/sec and we render $T$ triangles:
$B \approx \text{Pixels} \times \text{Textures} \times 4 \times T \times F$.

**Comparison:**
Advertised bandwidths for high-end GPUs (like the GeForce 9800 GTX mentioned) are around 70 GB/s. Achieved application bandwidth is typically lower (60-80% of peak) due to protocol overhead and DRAM efficiency.
