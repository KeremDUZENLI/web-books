### Answer 23.2
    
**Proof:**
First interpolation (u-direction first):
$T(u, 0) = (1-u)P_{00} + uP_{10}$
$T(u, 1) = (1-u)P_{01} + uP_{11}$
$S'(u, v) = (1-v)T(u, 0) + vT(u, 1)$
$S'(u, v) = (1-v)[(1-u)P_{00} + uP_{10}] + v[(1-u)P_{01} + uP_{11}]$
$S'(u, v) = (1-u)(1-v)P_{00} + u(1-v)P_{10} + (1-u)vP_{01} + uvP_{11}$

Comparison with original $S(u,v)$:
$S(u,v) = (1-u)(1-v)P_{00} + u(1-v)P_{10} + (1-u)vP_{01} + uvP_{11}$

Conclusion:
$S'(u, v) = S(u, v)$. The order of interpolation does not matter; bilinear interpolation is symmetric.
