### Answer 22.2
    
**(a) Identity of Curves:**
The Bézier curve is defined as $\gamma(t) = (1-t)^3 P_1 + 3t(1-t)^2 P_2 + 3t^2(1-t) P_3 + t^3 P_4$.
Substitute the given expressions:
$P_1 = P$
$P_2 = P + \frac{1}{3}v$
$P_3 = Q - \frac{1}{3}w$
$P_4 = Q$

Terms grouping by $P, Q, v, w$:
* **For P:** $(1-t)^3 + 3t(1-t)^2 = (1-t)^2((1-t) + 3t) = (1-2t+t^2)(1+2t) = 1 + 2t - 2t - 4t^2 + t^2 + 2t^3 = 2t^3 - 3t^2 + 1$. (Matches Hermite $H_P$).
* **For Q:** $3t^2(1-t) + t^3 = 3t^2 - 3t^3 + t^3 = -2t^3 + 3t^2$. (Matches Hermite $H_Q$).
* **For v:** $3t(1-t)^2 (\frac{1}{3}) = t(1-2t+t^2) = t^3 - 2t^2 + t$. (Matches Hermite $H_v$).
* **For w:** $3t^2(1-t) (-\frac{1}{3}) = -t^2(1-t) = t^3 - t^2$. (Matches Hermite $H_w$).

Since the coefficients match the Hermite polynomials, the curves are identical.

**(b) Matrix Form $S$:**
We express the Bézier geometry $G_B$ in terms of Hermite geometry $G_H = [P, Q, v, w]$.
$P_1 = 1\cdot P + 0\cdot Q + 0\cdot v + 0\cdot w$
$P_2 = 1\cdot P + 0\cdot Q + \frac{1}{3}\cdot v + 0\cdot w$
$P_3 = 0\cdot P + 1\cdot Q + 0\cdot v - \frac{1}{3}\cdot w$
$P_4 = 0\cdot P + 1\cdot Q + 0\cdot v + 0\cdot w$

Arranging these columns into $S$:
$S = \begin{bmatrix} 1 & 1 & 0 & 0 \\ 0 & 0 & 1 & 0 \\ 0 & 1/3 & 0 & 0 \\ 0 & 0 & -1/3 & 1 \end{bmatrix}$
Thus $G_B = G_H S$.

**(c) Determination of $M_H$:**
Given $G_B M_B = G_H M_H$ and $G_B = G_H S$.
Substitute $G_B$:
$(G_H S) M_B = G_H M_H$
$G_H (S M_B) = G_H M_H$
Since this holds for any geometry $G_H$, the basis matrices must satisfy:
$M_H = S M_B$.
