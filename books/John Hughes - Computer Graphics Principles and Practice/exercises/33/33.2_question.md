### Exercise 33.2
    
Write a vertex shader that alters the x-coordinate of each surface point as a sinusoidal function of y and time (which you'll need to pass to the shader from the host program, which will get the time from the system clock). You'll write something like
`gl_Position.x += sin(k1 * gl_Position.y + k2 * t);`
which will produce waves of wavelength $\frac{2\pi}{k_{1}}$ moving with velocity $\frac{2\pi}{k_{2}}$.
