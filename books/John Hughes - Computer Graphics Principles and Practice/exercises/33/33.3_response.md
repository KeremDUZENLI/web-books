### Answer 33.3
    
To draw triangles in a flat color specified by the host, we pass the color as a `uniform` to the vertex shader, which then passes it to the fragment shader. Since the color is flat (constant for the triangle), we do not need to vary it per vertex, but the vertex shader is the entry point for attributes.

**Vertex Shader:**
```glsl
uniform vec3 flatColor; // Color specified by host
varying vec3 colorOut;  // Output to fragment shader

void main(void) {
    colorOut = flatColor;
    gl_Position = gl_ModelViewProjectionMatrix * gl_Vertex;
}

```

**Fragment Shader:**

```glsl
varying vec3 colorOut;

void main(void) {
    gl_FragColor = vec4(colorOut, 1.0);
}

```

*Note: In the host program, one would issue a draw call for a specific triangle (or batch of triangles) and set the `flatColor` uniform before that call.*
