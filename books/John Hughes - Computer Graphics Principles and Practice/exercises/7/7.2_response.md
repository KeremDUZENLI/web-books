### Answer 7.2

Given $Ax + By + C = 0$ and $n = [A, B]^T$.
We want a point $P = O + \alpha n = \alpha [A, B]^T = [\alpha A, \alpha B]^T$.
Substitute $P$ into the line equation:
$A(\alpha A) + B(\alpha B) + C = 0$
$\alpha(A^2 + B^2) + C = 0$
$\alpha = \frac{-C}{A^2 + B^2}$
Thus, the point is $P = \frac{-C}{A^2 + B^2} \begin{bmatrix} A \\ B \end{bmatrix}$.
