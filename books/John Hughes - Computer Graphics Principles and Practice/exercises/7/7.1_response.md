### Answer 7.1

**(a)**
$||sv|| = \sqrt{(sv_1)^2 + (sv_2)^2 + ... + (sv_n)^2}$
$= \sqrt{s^2v_1^2 + s^2v_2^2 + ... + s^2v_n^2}$
$= \sqrt{s^2(v_1^2 + v_2^2 + ... + v_n^2)}$
$= \sqrt{s^2}\sqrt{v_1^2 + v_2^2 + ... + v_n^2}$
$= |s| ||v||$

**(b)**
If $s = -2$ and $v = [1, 0]^T$:
$||sv|| = ||[-2, 0]|| = \sqrt{(-2)^2 + 0} = \sqrt{4} = 2$.
$s||v|| = -2 \cdot \sqrt{1^2 + 0} = -2 \cdot 1 = -2$.
Since $2 \ne -2$, $||sv|| \ne s||v||$.
