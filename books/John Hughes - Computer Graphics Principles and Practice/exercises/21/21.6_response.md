### Answer 21.6
    
**Non-Uniform Scaling (2-Finger):**
* **Standard (Uniform):** Preserves aspect ratio. Scale factor $s = d_{current} / d_{start}$.
* **Non-Uniform:**
    * Let vector $\vec{v}_{start} = P2_{start} - P1_{start} = (\Delta x_{start}, \Delta y_{start})$.
    * Let vector $\vec{v}_{curr} = P2_{curr} - P1_{curr} = (\Delta x_{curr}, \Delta y_{curr})$.
    * Compute separate scale factors: $s_x = \Delta x_{curr} / \Delta x_{start}$ and $s_y = \Delta y_{curr} / \Delta y_{start}$.
    * Apply scale transformation $S(s_x, s_y)$.
* **Contrast:**
    * **Uniform:** Moving fingers apart diagonally scales the whole photo.
    * **Non-Uniform:** Moving fingers apart horizontally stretches the photo wide. Moving vertically stretches it tall.
    * **User Experience:** Non-uniform scaling is rarely desired for photos (distorts content). It requires the user to be very precise if they *don't* want distortion. Uniform scaling (using the vector magnitude) is robust against accidental aspect ratio changes.
