### Answer 12.3
**Equivalence Proof:**
Let points be $A, B, C, D$.
**Condition (a):** No three points collinear.
**Condition (b):** $A, B, C$ form triangle; $D$ not on lines extended from $AB, BC, CA$.

1.  **Fail (a) $\rightarrow$ Fail (b):**
    * If $A, B, C$ collinear: Triangle is degenerate. Fails (b).
    * If $A, B, D$ collinear: $D$ lies on line $AB$. Fails (b).
    * Same for other combinations.

2.  **Fail (b) $\rightarrow$ Fail (a):**
    * If triangle degenerate: $A, B, C$ are collinear. Fails (a).
    * If $D$ on extension of $AB$: $A, B, D$ are collinear. Fails (a).

**Conclusion:** The set of geometric configurations excluded by (a) is exactly the set excluded by (b).
