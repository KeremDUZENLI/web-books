### Answer 3.8

**Implementation Logic:**

1.  **Event Handler:** `protected override void OnKeyDown(KeyEventArgs e)`
2.  **Input Check:** `if (e.Key == Key.W)`
3.  **Math:**
    * Current Position: $P_{old} = (x, y, z)$
    * Step Vector: $\mathbf{v} = (0, 0, -1)$ (assuming -Z is forward/into screen).
    * New Position: $P_{new} = (x, y, z - \text{stepSize})$
4.  **Update:** `Camera.Position = new Point3D(P_new.x, P_new.y, P_new.z)`

**Reverse ('S' Key):**
$P_{new} = (x, y, z + \text{stepSize})$.
