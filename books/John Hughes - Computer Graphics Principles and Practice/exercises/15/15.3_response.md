### Answer 15.3
    
The 2D barycentric weight function is linear in $x$ and $y$:
$E(x, y) = Ax + By + C$

**Optimized Inner Loop Structure:**
1.  **Pre-loop:** Compute edge function coefficients ($A_i, B_i, C_i$) for all three edges $i \in \{0, 1, 2\}$ based on triangle vertices.
2.  **Outer Loop (y):** Initialize row start values: $w_{i\_row} = A_i x_0 + B_i y + C_i$.
3.  **Inner Loop (x):**
    Increment weights: $w_i = w_{i\_prev} + A_i$.
    If ($w_0 > 0$ && $w_1 > 0$ && $w_2 > 0$):
        Compute reciprocal depth: $z_{inv} = 1.0 / (w_0 z_0 + w_1 z_1 + w_2 z_2)$. (Single division).
        Interpolate attributes using $w_i$ and $z_{inv}$.

This approach replaces full line-distance calculations with incremental additions and reduces the division count to one per covered pixel for perspective correction.
