### Answer 4.4
**Implementation:**
1.  **Group Setup:**
    ```csharp
    Transform3DGroup group = new Transform3DGroup();
    TranslateTransform3D translate = new TranslateTransform3D();
    RotateTransform3D rotate = new RotateTransform3D(
        new AxisAngleRotation3D(new Vector3D(0, 1, 0), 0)
    );
    // Order matters: typically Rotate then Translate for local spinning
    group.Children.Add(rotate);
    group.Children.Add(translate);
    torso.Transform = group;
    ```
2.  **Binding:**
    * Create `private double _robotRotation;`.
    * In `OnUpdate`: `((AxisAngleRotation3D)rotate.Rotation).Angle = _robotRotation;`.
    * Add slider for `_robotRotation`.
