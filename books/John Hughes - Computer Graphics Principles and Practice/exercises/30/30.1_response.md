### Answer 30.1
    
**(a) Expected Value $\mu$**
The function $f(x)$ takes values $A$ on $[0,1)$, $B$ on $[1,2)$, and $C$ on $[2,3]$.
The integral $I = \int_0^3 f(x) dx = 1\cdot A + 1\cdot B + 1\cdot C = A + B + C$.
The random variable $X$ is generated by picking $i \in \{1, 2, 3\}$ with probability $1/3$.
If $i=1$, sample is $f(0.5) = A$. Estimate $X = 3A$.
If $i=2$, sample is $f(1.5) = B$. Estimate $X = 3B$.
If $i=3$, sample is $f(2.5) = C$. Estimate $X = 3C$.
Expectation:
$$ E[X] = \frac{1}{3}(3A) + \frac{1}{3}(3B) + \frac{1}{3}(3C) = A + B + C $$
Thus, $\mu = I$, so the estimator is unbiased.

**(b) Variance of $X$**
$$ Var[X] = E[X^2] - (E[X])^2 $$
$$ E[X^2] = \frac{1}{3}(3A)^2 + \frac{1}{3}(3B)^2 + \frac{1}{3}(3C)^2 = 3(A^2 + B^2 + C^2) $$
$$ Var[X] = 3(A^2 + B^2 + C^2) - (A+B+C)^2 $$
Or in terms of $\mu$:
$$ Var[X] = \frac{1}{3} \sum (X_i - \mu)^2 = \frac{1}{3} [ (3A - \mu)^2 + (3B - \mu)^2 + (3C - \mu)^2 ] $$

**(c) Conditions**
**Zero Variance:** Variance is zero if $X$ is always the same value.
$3A = 3B = 3C \implies A = B = C$.
The function must be constant.

**Large Variance:** Variance is large if the values $A, B, C$ are very different (high dispersion). For example, if $A=1000, B=1, C=1$, the mean is 1002, but individual estimates are 3000, 3, and 3. The variance scales with the differences between $A, B, C$.
