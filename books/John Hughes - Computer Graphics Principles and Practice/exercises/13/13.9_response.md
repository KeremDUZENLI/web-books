### Answer 13.9
Matrix $M = \begin{bmatrix} a & b \\ c & d \end{bmatrix}$ acting on $[z, 1]^T$.
Result: $[az+b, cz+d]^T$. $w' = cz+d$.

**(a) Eyepoint Condition:**
Eye at $z=0$ (in unhinged space setup where eye is at origin of projection, usually $z=0$ in simplified 2D).
Input $(0, 1)$. Output $w' = c(0) + d = d$.
Condition $w'=0 \implies d=0$.

**(b) Fixed Plane Condition:**
$z=-1 \implies (-1, 1)$. Output should map to depth -1.
Result vector: $[-a+b, -c]^T$.
Depth = $z'/w' = \frac{-a+b}{-c}$.
Condition: $\frac{-a+b}{-c} = -1 \implies -a+b = c$.
Standard projection sets $w' = -z$, implying $c=-1$.
Substitute $c=-1$: $-a+b = -1 \implies a = b+1$.

**(c) Near/Far Condition:**
Plane $z = -n/f$ maps to $z=0$.
Input $(-n/f, 1)$.
Numerator $z' = a(-n/f) + b$.
Condition $z'=0 \implies -a\frac{n}{f} + b = 0 \implies b = a\frac{n}{f}$.
Substitute $a = b+1$:
$b = (b+1)\frac{n}{f} \implies bf = bn + n \implies b(f-n) = n$.
$b = \frac{n}{f-n}$.
