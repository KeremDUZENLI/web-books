### Answer 9.5
**(b) Equality:** $H(x,0)$ describes a path where the first argument loops around $S^1$ and the second is fixed. $H(2\pi, y)$ describes a path where the first is fixed and the second loops. Due to the symmetry condition $H(p,q) = H(q,p)$, the topological "winding" behavior along these axes must be identical.
**(c) Winding Number 1:** Along the diagonal from $(2\pi, 2\pi)$ to $(0,0)$, the function is $H(t,t)$. By the condition $H(p,p)=p$, as the input loops once around the circle (diagonal of the domain), the output must loop exactly once around $S^1$.
**(d) Odd Total:** The total winding number is the sum of the winding numbers of the path segments. If the first two legs contribute $N$ and $N$ (or $N$ and $-N$ depending on orientation conventions but having equal magnitude/parity), and the diagonal contributes 1, the total is $2N \pm 1$, which is always odd.
**(e) Constant Winding:** The winding number is a topological invariant (an integer) that cannot change continuously. As the triangular loop shrinks continuously toward the center, the winding number must remain constant.
**(f) Zero Winding:** When the loop shrinks to a single point, the image of the map is a single point. A constant map has a winding number of zero.
**(g) Contradiction:** The winding number cannot be both non-zero (odd) and zero. Thus, the continuous function $H$ cannot exist.
