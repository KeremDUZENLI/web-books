### Answer 25.4
    
**(a) Contour Detection Algorithm:**
1.  Initialize a hash table `EdgeTable` mapping edges (pairs of vertex indices, sorted) to a sign value.
2.  Iterate over all faces $f$ in the mesh:
    * Calculate the dot product $d = n_f \cdot v$.
    * Determine the sign $s = \text{sign}(d)$.
    * For each of the three edges $e$ of $f$:
        * If $e$ is not in `EdgeTable`, store $(e, s)$.
        * If $e$ is in `EdgeTable` with stored sign $s_{stored}$:
            * If $s \neq s_{stored}$, the edge connects a front-facing and back-facing polygon. Mark $e$ as a **contour edge**.
            * (Optional) Remove $e$ from table to save space if only boundary/contour edges are needed.

**(b) Loop Assembly $O(E)$:**
1.  Store all contour edges in an adjacency list (or hash map) `Adj` where `Adj[v]` contains the vertices connected to `v` by a contour edge.
2.  Initialize a set `Visited` for edges.
3.  Iterate through the list of contour edges. If edge $(u, w)$ is not `Visited`:
    * Start a new loop list.
    * Traverse the graph from $u$ to $w$, marking edges as `Visited`.
    * At each vertex, pick the next unvisited neighbor (contour loops are generally 1-manifolds, so degree is 2).
    * Continue until the traversal returns to $u$.
    * Store the completed loop.
This visits each edge exactly once, resulting in $O(E)$ time.
