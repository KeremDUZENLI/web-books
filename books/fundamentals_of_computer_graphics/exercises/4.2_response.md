- **Given:**

  - Ray $\mathbf{p}(t) = (1, 1, 1) + t(-1, -1, -1)$

  - Triangle vertices $\mathbf{a}=(1,0,0), \mathbf{b}=(0,1,0), \mathbf{c}=(0,0,1)$.

- **Plane Intersection:** The triangle lies on the plane $x + y + z = 1$.

  - Substitute ray: $(1-t) + (1-t) + (1-t) = 1 \Rightarrow 3 - 3t = 1 \Rightarrow 3t = 2 \Rightarrow t = 2/3$.

- **Intersection Point:** $\mathbf{p}(2/3) = (1/3, 1/3, 1/3)$.

- **Barycentric Coordinates:**

  - We need weights $\alpha, \beta, \gamma$ such that $\mathbf{p} = \alpha\mathbf{a} + \beta\mathbf{b} + \gamma\mathbf{c}$.

  - $(1/3, 1/3, 1/3) = \alpha(1,0,0) + \beta(0,1,0) + \gamma(0,0,1) = (\alpha, \beta, \gamma)$.

  - Thus, $\alpha = 1/3, \beta = 1/3, \gamma = 1/3$.

- **Result:** $t = 2/3$ and barycentric coordinates $(1/3, 1/3, 1/3)$. Since $\alpha, \beta, \gamma \ge 0$, the intersection is valid.
