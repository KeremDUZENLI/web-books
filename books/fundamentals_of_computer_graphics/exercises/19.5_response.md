- **Model:**

  - `Paddle`: Rectangle (controlled by user).

  - `Ball`: Circle (approximated as bounding square for collision).

  - `Velocity`: Vector $(v_x, v_y)$.

- **Controller:**

  - **Mouse Move:** Updates `Paddle` X-position.

  - **Timer Event:**

    1.  Update Ball Position: $\mathbf{p} += \mathbf{v} \Delta t$.

    2.  **Gravity:** $v_y += g \Delta t$.

    3.  **Wall Collision:** If ball hits left/right/top, reflect velocity (e.g., $v_x = -v_x$).

    4.  **Paddle Collision:** Use AABB test from Ex 4. If hit:

        - Reflect Y velocity ($v_y = -|v_y|$ to ensure it goes up).

        - (Optional) Adjust $v_x$ based on where it hit the paddle for gameplay control.

    5.  **Fail:** If ball Y > Screen Height, reset/game over.
