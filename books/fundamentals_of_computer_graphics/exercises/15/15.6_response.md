- **Curve:** $\mathbf{f}(u) = (u, u^2)$.

- **Derivative:** $\mathbf{f}'(u) = (1, 2u)$.

- **Speed (Magnitude):** $\|\mathbf{f}'(u)\| = \sqrt{1^2 + (2u)^2} = \sqrt{1 + 4u^2}$.

- **Arc Length Function $s(t)$:**

  $$s(t) = \int_0^t \sqrt{1 + 4u^2} \, du$$

  This integral evaluates to $s = \frac{1}{4}(2t\sqrt{1+4t^2} + \sinh^{-1}(2t))$.

- **Parameterization:** To parameterize by arc length, we find the inverse function $t(s)$ (usually numerically, as the inverse of the above function is complex) and define the curve as $\mathbf{g}(s) = \mathbf{f}(t(s))$.
