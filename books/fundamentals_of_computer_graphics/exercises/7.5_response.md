- **Scenario:** Transforming a point $(x, y, z, 1)$ vs. a scaled point $(hx, hy, hz, h)$ using a matrix where the last row is $(0, 0, 0, 1)$.

- **Standard Point:** Result is $(x', y', z', 1)$.

- **Scaled Point:**

  - Input: $h \cdot (x, y, z, 1)$.

  - Since matrix multiplication is linear, Output = $h \cdot (x', y', z', 1) = (hx', hy', hz', h)$.

- **Homogenization:** To convert back to Cartesian, divide by the $w$-component ($h$):

  $$(\frac{hx'}{h}, \frac{hy'}{h}, \frac{hz'}{h}) = (x', y', z')$$

- **Result:** The physical 3D point is identical.
