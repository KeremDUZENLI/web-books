- **Definition:** A matrix $\mathbf{M}$ is symmetric if $\mathbf{M} = \mathbf{M}^T$.

- **Proof:** Let $\mathbf{M} = \mathbf{A}\mathbf{A}^T$.
  Take the transpose: $(\mathbf{A}\mathbf{A}^T)^T$.
  Using the property $(\mathbf{X}\mathbf{Y})^T = \mathbf{Y}^T \mathbf{X}^T$, we get $(\mathbf{A}^T)^T \mathbf{A}^T$.
  Since $(\mathbf{A}^T)^T = \mathbf{A}$, this simplifies back to $\mathbf{A}\mathbf{A}^T$.

- **Result:** The transpose equals the original, so it is symmetric.
