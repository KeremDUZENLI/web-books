- **Application State:**

  - `CurrentLine`: Structure holding `Center` (Point) and `EndPoint` (Point).

  - `IsDragging`: Boolean flag.

- **Event Handlers:**

  - **LMB Down:**

    - Set `CurrentLine.Center = MousePosition`.

    - Set `CurrentLine.EndPoint = MousePosition`.

    - Set `IsDragging = true`.

  - **LMB Drag:**

    - If `IsDragging`:

      - Update `CurrentLine.EndPoint = MousePosition`.

      - Visualizing `Draw line from `Center`to`EndPoint`AND from`Center`to`(Center - (EndPoint - Center))` (symmetry).`

  - **LMB Up:**

    - Set `IsDragging = false`.

    - Commit line to application database.

  - **RMB Drag:**

    - Calculate `Delta = MousePosition - LastMousePosition`.

    - Update `CurrentLine.Center += Delta`.

    - Update `CurrentLine.EndPoint += Delta` (maintains length/orientation).

- **Implementation Note:** This follows the **Event-Driven** model where the "drag" state persists between events.
