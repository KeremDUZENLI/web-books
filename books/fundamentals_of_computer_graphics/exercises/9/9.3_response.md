- **Concept:** A B-spline of degree $n$ is formed by convolving a box function (degree 0) with itself $n$ times.

- **Progression:**

  1.  **Box ($B_0$):** Value is 1 over interval $[-0.5, 0.5]$. Width = 1.

  2.  **Tent ($B_1 = B_0 \star B_0$):** A triangle. Width = 2.

  3.  **Quadratic ($B_2 = B_1 \star B_0$):** Smooth bell-like shape made of parabolas. Width = 3.

  4.  **Cubic ($B_3 = B_2 \star B_0$):** The standard cubic B-spline. Width = 4.

- **Proof Logic:**

  $$f_B(x) = f_{box} \star f_{box} \star f_{box} \star f_{box}$$

  Since the box function is the Fourier transform pair of $\text{sinc}(u)$, and convolution in spatial domain equals multiplication in frequency domain:

  $$
  \mathcal{F}\{B_3\} = (\mathcal{F}\{B_0\})^4 = (\text{sinc}(u))^4
  $$

  This confirms the B-spline property defined in the text.
